# 作業環境評価とは？？管理区分判定

作業環境評価は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、作業環境評価の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 評価の流れ

| 段階 | 内容 |
| --- | --- |
| 測定 | A測定・B測定 |
| 計算 | 評価値を求める |
| 比較 | 管理濃度と比べる |
| 判定 | 管理区分を決める |
| 措置 | 必要な改善 |

## 管理区分

| 区分 | 意味 |
| --- | --- |
| 第1 | 適切 |
| 第2 | 改善余地 |
| 第3 | 不適切・直ちに改善 |

## 試験で狙われる頻出ポイント

- 第1管理区分は良好、第3管理区分は直ちに改善が必要という順序を確実に覚える。
- A測定は作業場の平均的状態、B測定は高濃度になりやすい地点をみる考え方。
- 評価後は区分に応じて設備改善、作業方法改善、再測定、周知を行う。

---

## 試験での見方を整理する

作業環境評価は、定義だけでなく条件の言い換えまで問われやすいテーマです。本文の表で対象・頻度・数値・義務の主体を確認し、頻出ポイントで誤り選択肢を判断できるようにします。
