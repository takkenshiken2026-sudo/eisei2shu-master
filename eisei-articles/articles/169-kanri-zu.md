# 管理図とは？？区域・測定点・掲示

管理図は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、管理図の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 管理図に入れる情報

| 項目 | 目的 |
| --- | --- |
| 発散源 | 有害物の発生位置 |
| 測定点 | 評価の根拠 |
| 換気設備 | 気流と排気経路 |
| 避難経路 | 緊急時の退避 |

## 試験での関連

| 論点 | 結び付く事項 |
| --- | --- |
| 作業環境測定 | 単位作業場所・測定点 |
| 標識掲示 | 入口・危険区域 |
| 改善措置 | 局所排気や隔離の位置 |

## 試験で狙われる頻出ポイント

- 測定点や発散源の位置を図示すると、A測定・B測定や改善箇所を説明しやすい。
- 管理区域や立入禁止範囲は、現場の標識と一致させる必要がある。
- 設備変更や作業変更があれば、管理図も更新しないと古い情報になる。

---

## 試験での見方を整理する

管理図は、定義だけでなく条件の言い換えまで問われやすいテーマです。本文の表で対象・頻度・数値・義務の主体を確認し、頻出ポイントで誤り選択肢を判断できるようにします。

---

## 次に確認すること

管理図を確認したら、関連用語・まとめ記事・過去問へ進み、似た制度は横並びで比較してください。受験前には公式情報もあわせて確認してください。
