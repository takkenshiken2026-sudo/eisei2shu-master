# 衝撃音とは？？ピーク音・騒音対策

衝撃音は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、衝撃音の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 騒音の比較

| 種類 | 特徴 |
| --- | --- |
| 連続騒音 | 持続的 |
| 衝撃音 | 瞬間的ピーク |
| 間欠騒音 | 断続的 |
| 変動騒音 | 時間で大きく変動 |

## 対策の方向

| 対策 | 例 |
| --- | --- |
| 発生源 | 打撃力低減・緩衝材 |
| 伝搬経路 | 遮音壁・防音囲い |
| 作業管理 | 距離・時間短縮 |
| 保護具 | 耳栓・イヤーマフ |

## 試験で狙われる頻出ポイント

- 衝撃音は短時間でも大きなピークを持つため、平均値だけで軽視しない。
- ハンマー打撃やプレス音では、防音囲い、緩衝材、作業方法変更が対策になる。
- 耳栓・イヤーマフは有効だが、発生源の低騒音化や遮音を先に検討する。

---

## 試験での見方を整理する

衝撃音は、定義だけでなく条件の言い換えまで問われやすいテーマです。本文の表で対象・頻度・数値・義務の主体を確認し、頻出ポイントで誤り選択肢を判断できるようにします。

---

## 次に確認すること

衝撃音を確認したら、関連用語・まとめ記事・過去問へ進み、似た制度は横並びで比較してください。受験前には公式情報もあわせて確認してください。
