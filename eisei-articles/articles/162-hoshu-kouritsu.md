# 捕集効率とは？？測定誤差・局所排気

捕集効率は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、捕集効率の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 使われる場面

| 場面 | 捕集効率の意味 |
| --- | --- |
| 作業環境測定 | 媒体に保持された割合 |
| フード | 発散源から吸い込めた割合 |
| 除じん装置 | 粒子を除去できた割合 |

## 低下要因

| 要因 | 例 |
| --- | --- |
| 流量不適 | 速すぎて吸収不足 |
| 破過 | 吸着管の容量超過 |
| 粒径 | 微粒子の通過 |
| 漏れ | 接続部の不良 |

## 試験で狙われる頻出ポイント

- 捕集効率が低いと実際の濃度を過小評価するため、捕集媒体と対象物の組合せが重要。
- 局所排気ではフードに入る割合、除じん装置では粒子を除去する割合として使われる。
- 100%を前提にしない。破過、漏れ、粒径、流速などで低下する点が問われる。

---

## 試験での見方を整理する

捕集効率は、定義だけでなく条件の言い換えまで問われやすいテーマです。本文の表で対象・頻度・数値・義務の主体を確認し、頻出ポイントで誤り選択肢を判断できるようにします。
