# 検知管とは？？ガス濃度の簡易測定

検知管は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、検知管の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 検知管測定の手順

| 手順 | 内容 |
| --- | --- |
| 選定 | 対象ガス・測定範囲を確認 |
| 準備 | 両端を折りポンプに接続 |
| 吸引 | 指定回数だけ吸引 |
| 読取 | 変色長と補正表で濃度 |

## 注意点

| 項目 | 誤差要因 |
| --- | --- |
| 温湿度 | 変色反応に影響 |
| 妨害ガス | 似た反応で高めに出る |
| 期限 | 試薬劣化 |
| ポンプ | 漏れで採気量不足 |

## 試験で狙われる頻出ポイント

- 検知管は対象ガスごとに選ぶ。別物質用を使うと濃度を正しく読めない。
- 指定された吸引回数や採気量を守る。ポンプ漏れやストローク不良は誤差になる。
- 変色境界の読み取り、温度補正、妨害ガスを確認し、正式測定と混同しない。

## 試験での見方を整理する

検知管は、定義だけでなく条件の言い換えまで問われやすいテーマです。本文の表で対象・頻度・数値・義務の主体を確認し、頻出ポイントで誤り選択肢を判断できるようにします。
