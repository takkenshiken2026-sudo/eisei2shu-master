# 吸引採取とは？？ポンプ・流量校正

吸引採取は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、吸引採取の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 吸引採取の手順

| 手順 | 内容 |
| --- | --- |
| 準備 | 媒体とポンプを接続 |
| 校正 | 流量を設定・記録 |
| 採取 | 所定時間吸引 |
| 終了 | 再校正し採気量を算出 |

## 捕集媒体の例

| 媒体 | 対象 |
| --- | --- |
| ろ紙 | 粉じん・金属 |
| 吸収液 | ガス |
| 吸着管 | 有機溶剤蒸気 |
| インピンジャー | 液捕集 |

## 試験で狙われる頻出ポイント

- 採気量は流量×採取時間。流量の単位L/minと時間の換算ミスに注意する。
- 採取前後に流量を確認し、大きくずれた場合は測定値の信頼性を検討する。
- ポンプ、チューブ、捕集管の接続漏れは濃度を過小評価する原因になる。

---

## 試験での見方を整理する

吸引採取は、定義だけでなく条件の言い換えまで問われやすいテーマです。本文の表で対象・頻度・数値・義務の主体を確認し、頻出ポイントで誤り選択肢を判断できるようにします。
