# 保護メガネとは？？飛沫・粉じん・側板

保護メガネは、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、保護メガネの意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 眼保護具の比較

| 種類 | 向くリスク |
| --- | --- |
| 側板付き眼鏡 | 切削片・軽い飛散 |
| ゴグル型 | 液体飛沫・粉じん |
| フェイスシールド | 顔面全体への飛散 |

## 作業別の選定

| 作業 | 確認する保護 |
| --- | --- |
| 薬液移し替え | ゴグルと耐薬品手袋 |
| 研磨 | 飛来物と粉じん対策 |
| 高圧洗浄 | 顔面飛沫と滑り対策 |

## 試験で狙われる頻出ポイント

- 側方からの飛散がある作業では、普通の眼鏡ではなく側板付きやゴグル型を選ぶ。
- フェイスシールドは顔面保護に有用だが、密閉性が必要な飛沫では保護メガネと併用する。
- 酸・アルカリの飛散では、保護メガネだけでなく洗眼設備と速やかな洗浄手順も重要。
