# 振動の評価とは？？手腕振動・作業時間

振動の評価は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、振動の評価の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 振動ばく露の要素

| 要素 | 内容 |
| --- | --- |
| 大きさ | 振動加速度 |
| 時間 | 1日の使用時間 |
| 周波数 | 手腕への影響 |
| 姿勢 | 握り力・押付け力 |

## 対策の例

| 対策 | 内容 |
| --- | --- |
| 工具選定 | 低振動型へ |
| 整備 | 刃先交換・バランス調整 |
| 作業管理 | 連続作業を短くする |
| 保護具 | 防振手袋を補助的に使用 |

## 試験で狙われる頻出ポイント

- 手腕振動では、振動加速度だけでなく使用時間が健康影響を左右する。
- 防振手袋だけでなく、低振動工具、整備、作業時間短縮、休憩が対策になる。
- 騒音と同様に、発生源対策と作業管理を優先し、保護具だけに頼らない。
