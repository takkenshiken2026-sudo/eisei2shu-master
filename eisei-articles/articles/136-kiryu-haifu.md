# 気流分布とは？？給排気・局所排気

気流分布は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、気流分布の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 気流を見るポイント

| 場所 | 確認 |
| --- | --- |
| 給気口 | 清浄空気の流入方向 |
| 発散源 | 汚染気流の広がり |
| 呼吸域 | 作業者が吸い込む経路 |
| 排気口 | 滞留部から排出できるか |

## 悪い気流の例

| 状態 | 問題 |
| --- | --- |
| 横風 | フード捕集を妨害 |
| 短絡 | 給気がすぐ排気へ |
| 滞留 | 低所や隅に高濃度 |
| 逆流 | 汚染空気が作業者側へ |

## 試験で狙われる頻出ポイント

- 発散源から作業者の呼吸域へ流れる気流は不適切。フード側へ流す配置が基本。
- 給気の吹出しが強すぎると、局所排気の捕集気流を乱して制御風速不足を招く。
- 煙管やスモークテスターで気流を見える化し、滞留やショートサーキットを確認する。

---

## 試験での見方を整理する

気流分布は、定義だけでなく条件の言い換えまで問われやすいテーマです。本文の表で対象・頻度・数値・義務の主体を確認し、頻出ポイントで誤り選択肢を判断できるようにします。
