# 認定機関とは？？作業環境測定の外部機関

認定機関は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、認定機関の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 依頼時に伝える情報

| 項目 | 理由 |
| --- | --- |
| 使用物質 | 測定方法の選定 |
| 作業時間 | 測定時刻の設定 |
| 換気設備 | 評価の前提 |
| 発散源 | 測定点の決定 |

## 結果後の対応

| 結果 | 事業者の対応 |
| --- | --- |
| 第1管理 | 維持管理 |
| 第2管理 | 改善検討 |
| 第3管理 | 直ちに改善 |
| 不明点 | 再測定・追加調査 |

## 試験で狙われる頻出ポイント

- 外部機関に依頼しても、作業環境を改善し労働者へ周知する責任は事業者に残る。
- 測定対象、単位作業場所、作業内容を正確に伝えないと、代表性の低い測定になる。
- 結果報告書は管理区分、測定条件、改善提案を確認し、記録として保存する。

---

## 試験での見方を整理する

認定機関は、定義だけでなく条件の言い換えまで問われやすいテーマです。本文の表で対象・頻度・数値・義務の主体を確認し、頻出ポイントで誤り選択肢を判断できるようにします。

---

## 次に確認すること

認定機関を確認したら、関連用語・まとめ記事・過去問へ進み、似た制度は横並びで比較してください。受験前には公式情報もあわせて確認してください。
