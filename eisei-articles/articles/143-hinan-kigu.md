# 避難器具とは？？救出・二次災害防止

避難器具は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、避難器具の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 器具の例

| 器具 | 用途 |
| --- | --- |
| 命綱・ハーネス | 入槽者の引き上げ |
| 送気マスク | 救助者の呼吸保護 |
| 担架 | 搬送 |
| 酸素濃度計 | 救助前の環境確認 |

## 救助手順の要点

| 段階 | 注意 |
| --- | --- |
| 発見 | 単独で飛び込まない |
| 確認 | 酸素・ガス濃度を測定 |
| 救助 | 保護具と命綱を使用 |
| 搬送 | 新鮮空気下で応急処置 |

## 試験で狙われる頻出ポイント

- 酸欠場所へ無保護で救助に入ると二次災害になる。救助者にも送気マスク等が必要。
- 命綱やハーネスは使用前に損傷、長さ、固定点を点検する手順が問われる。
- 避難器具は置くだけでなく、作業者が使える位置、訓練、連絡体制まで必要になる。
