# 血中濃度とは？？体内ばく露指標

血中濃度は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、血中濃度の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 濃度指標の比較

| 指標 | 何を表すか |
| --- | --- |
| 血中濃度 | 体内に入った量の一部 |
| 尿中濃度 | 排泄された代謝物等 |
| 作業環境濃度 | 作業場の空気中濃度 |

## 結果に影響する因子

| 因子 | 影響 |
| --- | --- |
| 吸入量 | 呼吸域濃度と作業時間 |
| 経皮吸収 | 手袋・皮膚接触 |
| 代謝 | 個人差・半減期 |
| 採血時期 | ピークと消失 |

## 試験で狙われる頻出ポイント

- 血中濃度は管理濃度と同じ単位・意味ではない。場の評価と体内評価を分ける。
- 経皮吸収が大きい物質では、空気濃度が低くても血中濃度が上がる可能性がある。
- 短半減期の物質は採血タイミングの影響が大きく、結果解釈で問われる。
- 「管理濃度と同じ指標」→ 作業環境と生体試料は別

---

## 試験での見方を整理する

血中濃度は、定義だけでなく条件の言い換えまで問われやすいテーマです。本文の表で対象・頻度・数値・義務の主体を確認し、頻出ポイントで誤り選択肢を判断できるようにします。
