# 化学防護服とは？？液密・気密・透過

化学防護服は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、化学防護服の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 防護服の区分

| 区分 | 試験での見方 |
| --- | --- |
| 気密型 | ガス・蒸気の侵入防止 |
| 液密型 | 液体の浸入防止 |
| スプレー防護 | 噴霧・飛沫対策 |
| 粉じん防護 | 粒子付着の低減 |

## 使用手順

| 段階 | 確認 |
| --- | --- |
| 選定 | SDSと透過時間 |
| 着用 | 袖口・面体接続 |
| 作業 | 破れ・熱負担 |
| 脱衣 | 汚染面に触れない |

## 試験で狙われる頻出ポイント

- 高毒性や蒸気ばく露では、防護服だけでなく送気マスクなど呼吸保護を合わせて選ぶ。
- 液密と気密は違う。液体飛沫に強くてもガス・蒸気に十分とは限らない。
- 化学防護服は熱ストレスを増やすため、WBGT、作業時間、休憩も管理対象になる。

化学防護服は、定義だけでなく条件の言い換えまで問われやすいテーマです。本文の表で対象・頻度・数値・義務の主体を確認し、頻出ポイントで誤り選択肢を判断できるようにします。
