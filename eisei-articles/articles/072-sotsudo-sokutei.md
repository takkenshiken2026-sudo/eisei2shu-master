# 騒音の測定とは？？A特性・等価騒音

騒音の測定は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、騒音の測定の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 測定の要素

| 項目 | 内容 |
| --- | --- |
| 測定器 | 騒音計 |
| 補正 | A特性 |
| 評価量 | 等価騒音レベル等 |
| 位置 | 作業者の耳に近い場所 |
| 時間 | 通常作業を代表する時間 |

## 騒音対策

| 段階 | 例 |
| --- | --- |
| 発生源 | 低騒音機械・整備 |
| 伝搬 | 遮音・吸音 |
| 作業管理 | 時間短縮・距離 |
| 保護具 | 耳栓・イヤーマフ |
| 健康管理 | 聴力検査 |

## 試験で狙われる頻出ポイント

- 騒音計はA特性で測ることが多く、dB(A)の意味を押さえる。
- 作業者の耳の位置に近い高さ・場所で、通常作業時の騒音を測る考え方が重要。
- 測定結果は設備対策、遮音、作業時間短縮、耳栓、聴力検査とつなげる。
