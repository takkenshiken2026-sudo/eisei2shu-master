# リスクアセスメントとは？？化学物質管理

リスクアセスメントは、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、リスクアセスメントの意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 基本手順

| 手順 | 内容 |
| --- | --- |
| 特定 | 危険性・有害性を把握 |
| 見積り | 重篤度とばく露可能性 |
| 低減 | 優先順位に沿って措置 |
| 記録 | 結果と措置を周知 |

## 低減措置の優先順位

| 順位 | 例 |
| --- | --- |
| 1 | 有害性の低い物質へ代替 |
| 2 | 密閉化・自動化 |
| 3 | 局所排気 |
| 4 | 作業手順・時間短縮 |
| 5 | 保護具 |

## 試験で狙われる頻出ポイント

- SDSやラベルから危険有害性を把握し、作業内容とばく露経路を洗い出す。
- 低減措置は代替、密閉化、局所排気、作業管理、保護具の順に優先して考える。
- 実施後は記録、労働者への周知、措置の有効性確認までが一連の流れになる。

## 試験での見方を整理する

リスクアセスメントは、定義だけでなく条件の言い換えまで問われやすいテーマです。本文の表で対象・頻度・数値・義務の主体を確認し、頻出ポイントで誤り選択肢を判断できるようにします。
