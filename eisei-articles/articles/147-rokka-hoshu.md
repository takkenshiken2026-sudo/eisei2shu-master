# ろ過捕集とは？？粉じん・フィルター

ろ過捕集は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、ろ過捕集の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 捕集対象

| 対象 | 方法 |
| --- | --- |
| 粉じん | ろ過捕集 |
| 金属ヒューム | ろ過捕集 |
| 有機溶剤蒸気 | 吸着管など |
| 水溶性ガス | 液捕集 |

## 手順の注意

| 段階 | 確認 |
| --- | --- |
| 前処理 | フィルター秤量・保管 |
| 採取 | 一定流量で吸引 |
| 終了 | 流量確認 |
| 分析 | 重量または成分分析 |

## 試験で狙われる頻出ポイント

- ろ過捕集は粒子状物質向け。ガス・蒸気の液捕集や吸着捕集と使い分ける。
- 採取前後のフィルター重量差を使う場合、湿度や静電気による秤量誤差に注意する。
- フィルターの目詰まりは流量低下を起こし、採気量の計算に影響する。
