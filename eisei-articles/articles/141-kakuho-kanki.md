# 確保換気とは？？必要換気量・換気回数

確保換気は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、確保換気の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 換気計算の基本

| 項目 | 式・見方 |
| --- | --- |
| 換気回数 | 換気量(m3/h)÷室容積(m3) |
| 必要換気量 | 発散量÷許容濃度差の考え方 |
| 希釈換気 | 有害物を全体で薄める |

## 換気方式の使い分け

| 場面 | 優先する対策 |
| --- | --- |
| 発散源が明確 | 局所排気 |
| 低毒性・少量発散 | 全体換気・確保換気 |
| 酸欠のおそれ | 換気と濃度測定 |

## 試験で狙われる頻出ポイント

- 換気回数は1時間に室内空気が何回入れ替わるかを示し、換気量÷室容積で考える。
- 必要換気量は発散量を目標濃度以下に薄める考え方で、局所排気の代替とは限らない。
- 給気不足では排気量が出ず、扉の開閉や隣室からの汚染流入も起きる。

確保換気は、定義だけでなく条件の言い換えまで問われやすいテーマです。本文の表で対象・頻度・数値・義務の主体を確認し、頻出ポイントで誤り選択肢を判断できるようにします。
