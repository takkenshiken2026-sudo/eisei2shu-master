# 酸素濃度計とは？？18%未満・酸欠確認

酸素濃度計は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、酸素濃度計の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 酸素濃度の目安

| 酸素濃度 | 評価 |
| --- | --- |
| 約21% | 通常空気 |
| 18%未満 | 酸素欠乏 |
| 低下傾向 | 換気・退避を検討 |

## 測定手順

| 段階 | 確認 |
| --- | --- |
| 校正 | ゼロ・スパン確認 |
| 立入前 | 上中下を測る |
| 作業中 | 連続監視や再測定 |
| 異常時 | 退避・換気・救助体制 |

## 試験で狙われる頻出ポイント

- 酸素欠乏は空気中酸素濃度18%未満。21%前後が通常空気という数値を押さえる。
- マンホールやタンクは上・中・下で濃度が違うことがあり、複数位置で測る。
- 可燃性ガス計や硫化水素計とは測るものが違うため、必要に応じて併用する。

## 試験での見方を整理する

酸素濃度計は、定義だけでなく条件の言い換えまで問われやすいテーマです。本文の表で対象・頻度・数値・義務の主体を確認し、頻出ポイントで誤り選択肢を判断できるようにします。
