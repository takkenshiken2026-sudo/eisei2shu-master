# 有機溶剤作業主任者とは？？技能講習と作業指揮

有機溶剤作業主任者は、第二種衛生管理者試験の関係法令で頻出の用語です。この記事では、有機溶剤作業主任者の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 要点整理

| 観点 | 内容 |
| --- | --- |
| 用語 | 有機溶剤作業主任者 |
| 分野 | 関係法令 |
| 要点 | 有機溶剤作業で、作業方法、換気装置、保護具使用などを管理・指揮する作業主任者です。 |

| よくある誤り | 正しい整理 |
| --- | --- |
| 衛生管理者が兼ねれば選任不要 | 作業主任者としての選任が必要 |

| 作業主任者は健診を行う | 健診実施者ではなく現場管理の職務 |
