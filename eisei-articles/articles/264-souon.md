# 騒音とは？？聴力障害・作業環境

騒音は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、騒音の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 騒音の評価

| 項目 | 内容 |
| --- | --- |
| 音圧レベル | dBで表す |
| A特性 | 聴感補正 |
| 等価騒音 | 時間平均的評価 |
| 衝撃音 | ピーク性を考慮 |

## 健康影響と対策

| 項目 | 例 |
| --- | --- |
| 健康影響 | 騒音性難聴・疲労 |
| 作業環境管理 | 低騒音機械・遮音 |
| 作業管理 | 作業時間短縮 |
| 保護具 | 耳栓・イヤーマフ |
| 健康管理 | 聴力検査 |

## 試験で狙われる頻出ポイント

- 騒音性難聴は内耳障害で、いったん進むと回復しにくい点が健康管理で問われる。
- 大きさだけでなく周波数と時間が影響するため、A特性や等価騒音レベルを使う。
- 対策は低騒音化、遮音、距離、時間管理、耳栓の順に優先して考える。
- 「騒音＝音の大きさのみ、周波数無関係」→ A特性補正等で評価
