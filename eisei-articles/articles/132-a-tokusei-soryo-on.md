# A特性騒音レベルとは？？dB(A)・騒音評価

A特性騒音レベルは、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、A特性騒音レベルの意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 騒音測定での位置づけ

| 項目 | 試験での見方 |
| --- | --- |
| A特性 | 人の聴感に合わせた周波数補正 |
| dB(A) | A特性補正後の騒音レベル |
| 等価騒音レベル | 変動騒音を時間平均的に評価 |

## 誤り肢の典型

| 表現 | 正しい整理 |
| --- | --- |
| 低周波も同じ重み | A特性では周波数により補正される |
| 瞬間値だけで管理 | 時間とばく露を含めて評価 |
| 保護具だけで完結 | 工学的対策が優先 |

## 試験で狙われる頻出ポイント

- 単位は一般にdB(A)。単なるdB表記やC特性との混同を狙う肢に注意する。
- 騒音ばく露の評価では瞬間最大値だけでなく、等価騒音レベルや測定時間との関係で読む。
- 85dB・90dBなどの管理目安は、耳栓配布だけでなく発生源対策、作業時間短縮、聴力検査とセットで整理する。

## 試験での見方を整理する

A特性騒音レベルは、定義だけでなく条件の言い換えまで問われやすいテーマです。本文の表で対象・頻度・数値・義務の主体を確認し、頻出ポイントで誤り選択肢を判断できるようにします。
