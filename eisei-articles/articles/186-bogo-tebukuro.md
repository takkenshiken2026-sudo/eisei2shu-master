# 防護手袋とは？？透過・浸透・材質

防護手袋は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、防護手袋の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 材質選定の目安

| 材質 | 試験での注意 |
| --- | --- |
| ニトリル | 油・一部溶剤に用いるが万能でない |
| ブチル | 一部ガス・ケトン類に強い場合がある |
| 天然ゴム | アレルギーと溶剤耐性に注意 |

## 用語の違い

| 用語 | 意味 |
| --- | --- |
| 貫通 | 穴や破れを通る |
| 浸透 | 継ぎ目やすき間から入る |
| 透過 | 材料内部を分子が通過する |

## 試験で狙われる頻出ポイント

- ラテックス、ニトリル、ネオプレン、ブチルなどは得意な物質が異なる。万能手袋はない。
- 化学防護では貫通、浸透、透過を区別する。見た目に穴がなくても透過でばく露する。
- 汚染手袋でドアノブや顔を触ると二次汚染になる。脱着手順と交換をセットで押さえる。

## 試験での見方を整理する

防護手袋は、定義だけでなく条件の言い換えまで問われやすいテーマです。本文の表で対象・頻度・数値・義務の主体を確認し、頻出ポイントで誤り選択肢を判断できるようにします。
