# 作業環境管理簿とは？？測定結果と改善記録

作業環境管理簿は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、作業環境管理簿の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 記載事項の例

| 項目 | 内容 |
| --- | --- |
| 測定日 | いつ測ったか |
| 測定場所 | 単位作業場所 |
| 結果 | 濃度・管理区分 |
| 措置 | 改善内容・実施日 |

## 関連記録

| 記録 | 関係 |
| --- | --- |
| 測定記録 | 濃度と条件 |
| 点検記録 | 局所排気の状態 |
| 教育記録 | 作業者への周知 |
| RA記録 | 化学物質リスク |

## 試験で狙われる頻出ポイント

- 測定値だけでなく、管理区分、実施日、測定条件、改善措置を残す点が重要。
- 第3管理区分の改善を実施した場合、再測定や措置内容を記録する。
- 記録は後日の健康障害防止や監督署対応に使うため、保存・閲覧性も問われる。

---

## 試験での見方を整理する

作業環境管理簿は、定義だけでなく条件の言い換えまで問われやすいテーマです。本文の表で対象・頻度・数値・義務の主体を確認し、頻出ポイントで誤り選択肢を判断できるようにします。
