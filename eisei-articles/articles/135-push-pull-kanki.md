# プッシュプル型換気とは？？送気と排気

プッシュプル型換気は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、プッシュプル型換気の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 方式の特徴

| 要素 | 役割 |
| --- | --- |
| プッシュ | 清浄空気で流れを作る |
| プル | 汚染空気を吸い込む |
| 作業域 | 流れの外または上流に置く |

## 適用と注意

| 場面 | 注意 |
| --- | --- |
| 塗装 | 均一な気流を作る |
| 開放槽 | 液面上を横流し |
| 横風 | 捕集を乱す |
| 給気過大 | 飛散・拡散を招く |

## 試験で狙われる頻出ポイント

- プッシュ側だけでは汚染空気を押し広げる危険があり、プル側の捕集とバランスが重要。
- 作業者を汚染気流の下流に置かない配置が図問題で問われる。
- 局所排気や囲い式フードが使いにくい広い発散面で選択肢になる。
