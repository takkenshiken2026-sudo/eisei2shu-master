# ミゼットインピンジャーとは？？液捕集器具

ミゼットインピンジャーは、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、ミゼットインピンジャーの意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 器具の構成

| 部分 | 役割 |
| --- | --- |
| 吸収液 | 対象ガスを溶かす・反応させる |
| ノズル | 気泡を作り接触面積を増やす |
| ポンプ | 一定流量で吸引 |
| 捕集瓶 | 液を保持する |

## 使用時の注意

| 項目 | 注意 |
| --- | --- |
| 液量 | 不足で捕集不良 |
| 流量 | 規定範囲を守る |
| 温度 | 吸収効率に影響 |
| 転倒 | 液の逆流・損失 |

## 試験で狙われる頻出ポイント

- 粉じん用のろ紙ではなく、ガス・蒸気を吸収液に取り込む器具として覚える。
- 吸引流量が大きすぎると吸収不足や液の飛散が起き、捕集効率が下がる。
- 採取後は吸収液を分析し、採気量と合わせて空気中濃度を求める。
