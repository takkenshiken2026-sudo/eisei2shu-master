# 認定衛生評価士とは？？評価と改善助言

認定衛生評価士は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、認定衛生評価士の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 関係者の役割

| 名称 | 主な役割 |
| --- | --- |
| 作業環境測定士 | 測定の実施・分析 |
| 測定機関 | 外部測定の実施 |
| 衛生評価の専門家 | 評価と改善助言 |
| 衛生管理者 | 職場の衛生管理 |

## 評価の観点

| 観点 | 確認内容 |
| --- | --- |
| 測定 | 管理濃度との関係 |
| 設備 | 局所排気・換気 |
| 作業 | ばく露時間・手順 |
| 保護具 | 選定と着用 |

## 試験で狙われる頻出ポイント

- 作業環境測定士は測定の専門資格として出題されやすく、評価士と名称を混同しない。
- 評価は測定値を見るだけでなく、発散源、換気、保護具、作業方法を総合して行う。
- 改善提案は管理区分やリスクアセスメント結果に基づき、優先順位を付ける。

## 試験での見方を整理する

認定衛生評価士は、定義だけでなく条件の言い換えまで問われやすいテーマです。本文の表で対象・頻度・数値・義務の主体を確認し、頻出ポイントで誤り選択肢を判断できるようにします。
