# 心理的負荷とは？？ストレスチェックと面接指導

| 分野 | 労働衛生 |
| 要点 | 仕事による精神的な負担を評価する概念で、ストレスチェック制度や面接指導と関連します。 |

| よくある誤り | 正しい整理 |
| --- | --- |
| 心理的負荷は主観だけで判断 | 制度上の検査・評価の枠組みがある |
