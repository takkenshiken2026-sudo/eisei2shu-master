# 局所排気装置の性能検査とは？？風量・制御風速

局所排気装置の性能検査は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、局所排気装置の性能検査の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 検査項目

| 項目 | 見る内容 |
| --- | --- |
| 制御風速 | フードで捕集できる速度 |
| 風量 | 必要な排気量 |
| 静圧 | 詰まり・抵抗の変化 |
| 気流 | 発散源からフードへ流れるか |

## 不良時の原因

| 症状 | 原因例 |
| --- | --- |
| 風速不足 | 排風機能力低下 |
| 騒音増加 | ダクト詰まり・振動 |
| 捕集漏れ | フード位置不良 |
| 粉じん飛散 | フィルター破損 |

## 試験で狙われる頻出ポイント

- フードの制御風速が不足すると、見た目に排気していても呼吸域へ漏れる。
- ダクト内の堆積、フィルター目詰まり、ベルト緩みは風量低下の原因になる。
- 設置時だけでなく、定期自主検査や補修後の確認、記録保存まで問われる。
