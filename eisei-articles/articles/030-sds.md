# SDSとは？？化学物質情報・保護具選定

SDS（安全データシート）は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、SDS（安全データシート）の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## SDSで見る項目

| 項目 | 使い道 |
| --- | --- |
| 危険有害性 | リスクの特定 |
| 応急措置 | 事故時対応 |
| ばく露防止 | 換気・保護具 |
| 物性 | 揮発性・引火性 |
| 廃棄 | 処理方法 |

## 関連する管理

| 管理 | SDSとの関係 |
| --- | --- |
| リスクアセスメント | 情報源 |
| 保護具選定 | 材質・種類を確認 |
| 教育 | 作業者へ周知 |
| ラベル | 容器表示と整合 |

## 試験で狙われる頻出ポイント

- SDSは入手して保管するだけでなく、作業者が理解できるよう周知・教育する。
- 保護具欄、ばく露防止欄、応急措置欄は、実際の作業手順に反映する。
- GHS分類やラベルの絵表示と対応し、危険有害性の特定に使う。
