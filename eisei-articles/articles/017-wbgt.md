# WBGTとは？？暑さ指数・熱中症予防

WBGTは、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、WBGTの意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## WBGTの式

| 条件 | 式 |
| --- | --- |
| 屋外・日射あり | 0.7Tnw + 0.2Tg + 0.1Ta |
| 屋内・日射なし | 0.7Tnw + 0.3Tg |
| Tnw | 自然湿球温度 |
| Tg | 黒球温度 |
| Ta | 乾球温度 |

## 熱中症予防

| 要素 | 内容 |
| --- | --- |
| 作業強度 | 代謝率が高いほど基準は厳しい |
| 順化 | 暑さに慣れていない人は注意 |
| 休憩 | 涼しい場所で計画的に取る |
| 補給 | 水分と塩分 |
| 衣服 | 化学防護服では熱負担増 |

## 試験で狙われる頻出ポイント

- 屋外で日射がある場合のWBGTは、0.7×自然湿球温度＋0.2×黒球温度＋0.1×乾球温度。
- 屋内または日射なしでは、0.7×自然湿球温度＋0.3×黒球温度で乾球温度を使わない。
- WBGT基準値は作業強度、暑熱順化、衣服で変わる。水分塩分、休憩、冷房を併用する。
