# 捕集分析法とは？？採気量・分析

捕集分析法は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、捕集分析法の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## 基本手順

| 段階 | 内容 |
| --- | --- |
| 校正 | 採取前後の流量確認 |
| 捕集 | 所定時間・所定位置で採取 |
| 分析 | 捕集量を定量 |
| 算出 | 捕集量÷採気量で濃度 |

## 捕集媒体

| 媒体 | 主な対象 |
| --- | --- |
| ろ紙 | 粉じん・金属ヒューム |
| 吸収液 | 水溶性ガス等 |
| 吸着管 | 有機溶剤蒸気 |
| バッグ | 一部ガスの瞬間採取 |

## 試験で狙われる頻出ポイント

- 濃度は分析で得た量を採気量で割って求めるため、流量と採取時間の記録が不可欠。
- 粉じんはろ過捕集、ガス・蒸気は液捕集や固体捕集など、対象物で方法を選ぶ。
- 検知管の直読値と異なり、捕集後の分析で確定値を出す手順を押さえる。

---

## 試験での見方を整理する

捕集分析法は、定義だけでなく条件の言い換えまで問われやすいテーマです。本文の表で対象・頻度・数値・義務の主体を確認し、頻出ポイントで誤り選択肢を判断できるようにします。
