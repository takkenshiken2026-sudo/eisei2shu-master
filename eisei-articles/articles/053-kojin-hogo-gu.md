# 個人用保護具とは？？PPE・選定順位

個人用保護具は、第二種衛生管理者試験の労働衛生で頻出の用語です。この記事では、個人用保護具の意味、試験で問われやすい観点、関連用語との違いを復習に使える形で整理します。

---

## PPEの種類

| 部位 | 保護具例 |
| --- | --- |
| 呼吸器 | 防毒マスク・送気マスク |
| 眼 | 保護メガネ・ゴグル |
| 手 | 防護手袋 |
| 体 | 防護衣・化学防護服 |
| 耳 | 耳栓・イヤーマフ |

## 対策の優先順位

| 順位 | 対策 |
| --- | --- |
| 1 | 有害性の低い物質へ代替 |
| 2 | 密閉化・自動化 |
| 3 | 局所排気・換気 |
| 4 | 作業管理 |
| 5 | 個人用保護具 |

## 試験で狙われる頻出ポイント

- 保護具は最後の砦であり、発散源対策や換気を省略する理由にはならない。
- 呼吸用、眼、手、体、聴覚など、ばく露経路ごとに必要な保護具を分ける。
- 選定、教育、着用確認、保守、交換を含めて管理しないと効果が落ちる。
