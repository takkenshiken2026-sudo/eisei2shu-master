第二種衛生管理者の出題傾向【30問·3科目·有害業務除外·new-topics-trend連携】

第二種衛生管理者試験の出題傾向は、30問·3科目×各10問·各科目40点足切り·有害業務除外が長年安定した骨格です。近年の頻出テーマ横断整理の正本は近年トレンド記事、本出題傾向記事は科目別の出方·演習への落とし方·44問情報の除外判断に特化します。試験は30問·300点·180点（60%）·各科目40点以上·約3時間（10月11日（日）13:30開始）·受験料8,800円です。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、第2週に本記事の3科目傾向表を転記し、第3週に近年トレンド記事で頻出3テーマを横並び確認するのが定番です。正本は安全衛生技術試験協会（https://www.exam.or.jp/）要項です。

この記事の要点

本記事を読むと、たとえば第3週の30問演習前に3科目×各10問の出題傾向表を10分で転記し、44問教材から拾った設問を範囲外として除外できるようになります。

1第二種30問の出題傾向骨格表

第二種の出題傾向はまず骨格を固定します。44問·100問は第一種の数値で、第二種は30問体系です。

| 項目 | 第二種（正） | 混同しやすい誤情報 |
| --- | --- | --- |
| 問題数 | 30問 | 44問·100問 |
| 科目 | 3科目×各10問 | 5科目 |
| 合格 | 180点·各40点 | 240点等 |
| 有害業務 | 出題外 | 第一種深掘り |
| 時間 | 約3時間·13:30 | — |

たとえば6月6日に「44問60%で勉強している」メモがある場合、本骨格表で30問·180点に修正するのが定番です。試験科目記事で3科目×各10問の配点も確認してください。正本はexam.or.jp要項です。

2new-topics-trend記事との役割分担

出題傾向関連は2記事で分担します。近年トレンド記事は要項·公式テキスト照合と科目横断の頻出テーマ·追加·削除論点が正本、本出題傾向記事は3科目別の出方·演習計画への落とし方·44問情報除外が正本です。

| 論点 | 近年トレンド記事 | 本出題傾向記事 |
| --- | --- | --- |
| 頻出一覧 | 横断整理 | — |
| 科目別出方 | 概要 | 詳細表 |
| 演習 | 30問形式 | 科目別10問 |
| 除外 | 有害業務 | 44問·100問 |
| 正本 | 要項·テキスト | 骨格表 |

  • 例えば第2週に本記事で3科目傾向表
  • 第3週に近年トレンド記事で関係法令·労働衛生·労働生理の頻出3テーマ
  • 第4週の30問演習で3科目40点以上
  • と順に読む流れが定番

出題範囲概要記事も併用してください。

33科目×各10問の出題傾向対照表

科目別の出題傾向を1枚表に並べると演習配分が決めやすくなります。

| 科目 | 10問の中心 | 頻出論点（代表） |
| --- | --- | --- |
| 関係法令 | 条文·数値 | 選任·健診·面接 |
| 労働衛生 | 方式·測定 | WBGT·局所排気 |
| 労働生理 | 因果·症状 | 疲労·温熱·振動 |
| 有害業務 | 出題外 | 除外ラベル |
| 演習 | 30問通し | 180分·13:30 |

たとえば第4週の30問演習で関係法令50点·労働衛生48点·労働生理36点なら、足切り不合格圏内です。第5〜7週は労働生理10問を週3回に増やすのが定番です。合格基準記事で180点·各40点の詳細も確認してください。7日後に弱点科目10問を解き直します。

4有害業務·第一種設問の除外フィルター

出題傾向を学ぶ際、範囲外設問を混ぜると得点記録が歪みます。除外ラベルを付けます。

| 設問タイプ | 第二種 | 対処 |
| --- | --- | --- |
| 基本選任·健診 | 範囲内 | 30問記録 |
| 特定化学物質深掘り | 範囲外 | 除外 |
| 44問形式 | 第一種 | 棚から外す |
| 100問5科目 | 旧表記 | 混同しない |
| 比較学習 | 可 | 得点記録除外 |

  • 例えば第5週の30問演習で第一種100問教材から拾った2問を誤答した場合
  • 本フィルター表で除外し
  • 第二種210問無料演習に切り替えるのが定番

第一種·第二種の違い記事で線引きも確認してください。過去問活用法記事で3段階解き方も併用できます。

5出題傾向を演習計画に反映する手順

傾向表は週次計画に変換して初めて効きます。18週逆算の具体手順です。

1. 骨格表を10分転記（30問·3科目·180点）
2. 近年トレンド記事で頻出3テーマ確認（20分）
3. 第1〜4週：科目別10問·テキスト並行
4. 第5週〜：日曜13:30·30問本番形式
5. 2週ごとに3科目40点以上·180点を記録

たとえば10月4日（日）13:30から30問演習で関係法令50点·労働衛生48点·労働生理44点·総合182点なら、出題傾向どおりの合格圏です。過去問活用法記事で年度別→科目別→30問の順も確認してください。受験料8,800円·各40点足切りは要項どおりです。

6よくある質問

近年トレンド記事との違いは？
近年トレンド記事は要項·公式テキスト照合と科目横断の頻出テーマ·追加·削除論点が正本、本出題傾向記事は3科目別の出方·演習計画への落とし方·44問·100問除外判断が正本です。たとえば第2週に本記事で3科目傾向表、第3週に近年トレンド記事で頻出3テーマ横並び、第4週の30問演習で3科目40点以上を記録、と順に読む流れが定番です。出題範囲概要記事で有害業務除外も確認してください。7日後に★印リストから範囲外問題を解き直し対象から外します。
2024年版表記は要項と矛盾しませんか？
出題傾向の数値·形式は要項·公式テキストが正本です。年度ラベル（2024年版等）は記事更新時期の目安であり、30問·180点·各40点·3科目×各10問·有害業務除外は要項で毎年確認してください。例えば6月6日に要項PDFを開き、試験科目·合格基準·出題範囲を1枚メモに転記し、本記事の骨格表と突合するのが定番です。公式情報確認先記事でexam.or.jp入口も確認できます。44問·100問は第一種の数値です。
44問教材は傾向把握に使えますか？
第一種44問·100問教材は第二種30問の出題傾向判断に使わないでください。比較学習として読むのは可ですが、得点記録·弱点分析には第二種30問演習だけを使います。たとえば第5週の30問演習で44問教材から拾った特定化学物質2問を誤答した場合、本記事の除外フィルター表で範囲外ラベルを付けるのが定番です。第一種·第二種の違い記事で線引きも確認してください。7日後に範囲内★印10問を解き直します。
