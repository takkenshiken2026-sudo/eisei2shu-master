第二種衛生管理者試験の関係法令の過去問での出題傾向

関係法令10問の過去問を分析すると、選任·健康診断·作業環境·長時間労働が毎回または隔年で登場し、ストレスチェック·安全衛生教育·罰則は相対的に出題頻度が低い傾向があります。第二種は30問·180点·各科目40点以上で、有害業務条文は出題範囲外です。たとえば残り12週の6月6日なら、直近5年分の関係法令10問（計50問）を12テーマ番号でタグ付けし、2回以上誤答した論点から条文復習に戻るのが定番です。本記事では過去問の焦点と分析手順を表に整理します。正本は安全衛生技術試験協会（https://www.exam.or.jp/）の過去問題と最新要項です。

この記事の要点

本記事を読むと、たとえば直近5年50問を「選任8問·健診12問·面接6問」と集計し、自分の演習誤答と過去問の焦点が一致する論点から優先復習できるようになります。

  • 過去問50問分析
  • 12テーマタグ
  • 選任·健診多め
  • 有害業務は出題外
  • 40点足切り

1関係法令10問の出題比率（目安）

直近年度を通した関係法令10問の傾向は次のとおりです（年度でブレますが、演習優先の目安になります）。

| 論点 | 10問中の目安 | 条文·制度 |
| --- | --- | --- |
| 選任·体制 | 2〜3問 | 法第13〜15条付近 |
| 健康診断 | 2〜3問 | 定期·特殊 |
| 作業環境 | 1〜2問 | 測定·評価 |
| 化学物質 | 1問前後 | 規則の適用 |
| 面接指導 | 1〜2問 | 80時間·主体 |
| その他 | 0〜1問 | 教育·罰則等 |

たとえば5年分50問を集計して「健診12問·選任10問·面接8問」となったら、健診条文の復習時間を週2時間確保します。全体の過去問進め方は過去問活用記事を参照してください。

2年度をまたぐ「繰り返し焦点」

同じ条文論点が年度を変えて再出題されることがあります。

| 焦点 | 問われ方の例 | 対策 |
| --- | --- | --- |
| 50人選任 | 人数境界 | 選任表 |
| 健診頻度 | 6ヶ月·1年 | 制度別整理 |
| 面接主体 | 産業医 vs 衛生管理者 | 主体表 |
| 保存期間 | 3年 vs 5年 | 数値リスト |
| 就業措置 | 事業者の判断 | ケース演習 |

  • 例えば2023年と2025年の過去問で「面接指導の実施者」が同型で出たなら
  • 2問を並べて誤答肢の共通パターン（主体混同）を1行メモし
  • 関係法令10問演習で同論点3問を解き直し

3過去問タグ付けの手順

関係法令10問だけを切り出して次の手順で分析します。

1. 直近5年×10問＝50問を一覧化
2. 各問に12テーマ番号（頻出論点記事）を付与
3. 正誤·誤答理由（主体/数値/制度）を記録
4. 2回以上誤答した番号を★優先
5. ★論点の条文をテキストで読み直す

たとえば50問中No.2健診を8問誤答·No.7面接を5問誤答と分類したら、第4週はNo.2·第5週はNo.7に週2時間ずつ配分します。12テーマの詳細は関係法令頻出論点記事を参照してください。

4出題されにくい論点（時間配分の参考）

すべての条文を同じ時間で読む必要はありません。

| 優先度 | 論点 | 過去問での位置づけ |
| --- | --- | --- |
| 高 | 選任·健診·面接 | ほぼ毎年 |
| 中 | 作業環境·化学 | 隔年〜毎年 |
| 低 | 罰則·立入のみ | 単発が多い |
| 出題外 | 有害業務 | 第二種対象外 |

  • 例えば残り4週で関係法令38点·未読章が罰則と有害業務だけなら
  • 有害業務章はスキップし
  • 選任·健診の過去問20問解き直しに週4時間を集中させる方が得点効率が高い

条文の骨格は関係法令基礎記事で確認してください。

5過去問分析から演習へつなぐ

分析の出口は「関係法令10問演習で40点以上2回連続」です。

| フェーズ | 内容 | 目標 |
| --- | --- | --- |
| 分析 | 50問タグ付け | ★論点5個以内 |
| 復習 | ★論点の条文 | テキスト該当章 |
| 演習 | 10問·本番形式 | 4問正解以上 |
| 確認 | 30問通し | 関係法令40点以上 |

  • たとえば第8週に過去問50問の分析を終え
  • ★論点が選任·健診·面接の3つに絞れたら
  • 第9週はこの3論点の一問一答15問→第10週に関係法令10問で5問正解（50点）を目標にし

学習時間の目安は勉強時間記事を参照してください。

6よくある質問

何年分の過去問を解けばよいですか？
関係法令に限れば直近5年分（50問）が傾向把握と足切り対策のバランスが良い目安です。たとえば6月6日開始·残り12週なら、第1〜4週で50問をタグ付け·第5週以降は★論点の演習10問を週2回、という配分が現実的です。全科目の過去問戦略は過去問活用記事を参照してください。
第一種の過去問は使えますか？
条文理解の補助には使えますが、出題数（100問）と有害業務の比重が異なります。例えば第一種100問の関係法令40問を第二種用にそのまま解くと時間を使いすぎます。第二種要項の出題範囲表で有害業務を除外し、12テーマに該当する設問だけをピックアップする使い方に留めてください。
過去問分析と頻出論点記事の違いは？
頻出論点記事は静的な優先表、本記事は自分の過去問データで優先順を更新する手順です。たとえば頻出論点で★★の面接指導が、自分の50問分析では1問しか出ていなくても、演習で2回連続誤答したなら自分の★リストでは最優先に上げてください。分析後は関係法令10問演習で得点を記録し続けます。
