第二種衛生管理者は何問正解で合格？科目別の合格ラインを問題数で計算

第二種衛生管理者試験の合格ラインは、総合18問正解相当（180点·300点中60%）かつ各科目4問正解相当（40点·100点中40%）の両方です（要項で再確認）。本記事は18問·各4問の正解数換算表·演習ログの正解数併記フォーマット·免除20問時の正解数目標に特化しています。詳細は各関連記事を参照してください。例えば10/4（日）に13:30から30問演習で総合19問（190を本記事の表で確認し、1週間後に解き直してください。

この記事の要点

本記事を読むと、たとえば9/14（日）30問演習で21問正解（210点）なのに労働生理3問正解（30点）で不合格になる理由が換算表で分かり、科目免除20問時の正解数目標も計算できます。

  • 18問=180点
  • 各4問=40点
  • 足切り両条件
  • 免除20問も同基準
  • 1問10点

1合格ラインの正解数換算（30問·300点）

第二種衛生管理者試験は30問·300点満点です（要項で再確認）。合格には次の2条件を同時に満たす必要があります。

条件得点正解数換算
総合180点以上18問以上/30問
労働衛生40点以上4問以上/10問
関係法令40点以上4問以上/10問
労働生理40点以上4問以上/10問
1問配点10点5肢択一

たとえば30問中18問正解（180点）·3科目すべて4問正解（各40点）がぎりぎり合格ラインです。19問正解（190点）·労働生理3問（30点）なら不合格です。足切り制度の詳細は合格基準記事をあわせて読んでください。44問·400点·240点は第一種の数値です。

2総合180点＝18問正解の読み方

総合点は正解数×10で心算できます。演習記録には得点と正解数の両方を書くと判断が速いです。

正解数得点判定目安
15問150点不合格
17問170点不合格
18問180点総合ぎり合格
20問200点余裕（足切り注意）
21問210点高得点でも足切りあり

例えば8/3（日）30問演習で20問正解（200点）·労働生理3問正解（30点）なら、総合は合格圏でも足切り不合格圏、と演習ログに赤字で書き込みます。8/10に労働生理10問を解き直し、4問以上を目標にしてください。有害業務は第二種出題外です。

3各科目40点＝4問正解/10問

科目別は10問中4問正解が足切りラインです（要項で再確認）。1科目9問正解（90点）でも他科目3問（30点）なら不合格です。

科目10問中得点あと1問の効果
関係法令4問40点+10点
労働衛生4問40点+10点
労働生理4問40点+10点
3問正解3問30点不合格圏
5問正解5問50点余裕

たとえば関係法令10問中3問正解（30点）なら、あと1問正解で40点に届きます。39点（3問正解+惜しい1問）の拾い方は惜しい問題見直し記事をあわせて読んでください。演習のたびに3科目の正解数を横並びで記録します。

4科目免除20問時の正解数目標

1科目（10問）免除で20問·おおむね2時間15分（135分）になります（要項で再確認）。合格基準180点·各40点は維持されます。

項目通常30問免除20問
問題数30問20問
満点300点200点
合格総合180点180点（18問相当）
科目数3科目2科目
各科目4問/10問4問/10問

例えば労働衛生免除で関係法令·労働生理の20問なら、総合12問正解（120点）では不足·各科目4問正解（各40点）·総合18問相当（180点）が必要、と換算表を6/18にノートへ書きます。申込手続の詳細は科目免除概要記事をあわせて読んでください。

5演習記録フォーマットと関連記事の2列比較

演習のたびに正解数を併記すると、18問·各4問ラインとの距離が一目で分かります。

1. 総合：○問正解（○点）/ 18問以上か
2. 関係法令：○問/10問 / 4問以上か
3. 労働衛生：○問/10問 / 4問以上か
4. 労働生理：○問/10問 / 4問以上か
5. 40点未満科目に「あと○問」を書く

論点本記事（正解数）合格点記事（得点）
換算18問·各4問表あと何問逆算
記録正解数併記得点シート
免除20問の正解数目標135分配分
制度概要のみ足切り→合格基準記事
演習例9/14（日）30問38点→39点拾い

たとえば10/4（日）13:30から30問演習で総合19問（190点）·関5問·衛5問·生4問なら、3科目すべて40点以上·総合も合格圏です。第一種44問60％の基準で記録をつけないでください。

6よくある質問

何問正解で合格ですか？
総合18問正解以上（180点）かつ、各科目4問正解以上（各40点）が必要です（要項で再確認）。たとえば30問中18問正解·3科目すべて4問正解がぎり合格ラインです。20問正解（200点）·労働生理3問（30点）なら不合格です。6/11に18問·各4問をノートへ書き、7/13の15問演習から正解数を併記してください。44問·60％は第一種のイメージと混同しないでください。
1科目だけ3問正解（30点）だと不合格ですか？
はい。1科目でも40点（4問正解）未満なら足切りで不合格です。たとえば総合21問正解（210点）·労働生理3問正解（30点）なら、総合点が高くても不合格になります。演習のたびに3科目の正解数を記録し、最も低い科目に週2時間以上を配分してください。9/7に弱点科目10問を解き直し、4問以上を目標にします。合格基準記事で足切りの制度説明も確認できます。
科目免除20問でも合格ラインは同じですか？
はい。免除後も総合180点·各科目40点以上が必要です（要項で再確認）。20問·満点200点でも180点（60%）·各10問中4問正解が基準です。例えば労働衛生免除で残り20問なら、2科目それぞれ4問以上·総合18問相当の得点配分を意識します。試験時間はおおむね2時間15分（135分）に短縮されます。科目免除概要記事で申込時申請·8,800円も確認してください。30問·3時間の通常形式と混同しないで記録してください。
