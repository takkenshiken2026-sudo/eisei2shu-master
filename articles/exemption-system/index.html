第二種衛生管理者試験の免除制度

第二種衛生管理者試験には、一定の資格保持者向けの科目免除制度があります。免除が認められると問題数が30問から20問に減り、試験時間もおおむね3時間から2時間15分（135分）に短縮されます。たとえば10月11日（日）13:30開始の回を科目免除付きで受ける場合、8月上旬の申込時に免除要件·証明書類を要項どおり提出する必要があります。正本は安全衛生技術試験協会（https://www.exam.or.jp/）の最新要項です。

この記事の要点

本記事を読むと、たとえば対象資格を持つ場合にどの科目が免除され、残り20問·135分で総合180点·各科目40点以上をどう満たすかが表で分かり、申込ミスや学習範囲の誤解を防げます。

  • 科目免除制度
  • 30問→20問
  • 135分
  • 申込時に申請
  • 要項が正本

1科目免除制度の概要

第二種衛生管理者試験の免除制度は、主に「科目免除」です。

一定の資格·学歴·業務経験を満たす受験者。

  • 1科目（10問）が免除され
  • 残り2科目20問を受験し

| 項目 | 通常 | 科目免除時 |
| --- | --- | --- |
| 問題数 | 30問 | 20問 |
| 時間 | おおむね3時間 | 2時間15分 |
| 受験料 | 8,800円 | 8,800円 |
| 形式 | 5肢択一 | 5肢択一 |

たとえば10月11日（日）13:30開始で免除適用なら、16:45頃終了のイメージです（会場案内に従ってください）。44問·100問は第一種の数値で、第二種は常に30問（免除時20問）です。

2免除対象者の典型例

免除対象は年度·要項で公表されるため、自分が該当する行を要項表で1つ特定します。医師·作業環境測定士等の資格保持者が典型例です。

| 確認 | 方法 |
| --- | --- |
| 対象資格 | 要項の免除表 |
| 免除科目 | 労働衛生等（年度で確認） |
| 証明書類 | 資格証·卒業証明等 |
| 申請期限 | 申込時·要項記載 |

  • 例えば6月6日時点で対象資格を取得済みなら
  • 8月上旬の申込開始までに証明書の写しを準備し
  • 科目免除記事で免除される科目名を確認してから申込み

3免除適用時の合格基準

科目免除があっても、合格基準は要項どおり確認が必要です。通常は総合180点（60％）以上·各科目40点（40％）以上が目安です。

| 基準 | 内容 |
| --- | --- |
| 総合 | 180点以上（300点満点換算） |
| 各科目 | 40点以上（足切り） |
| 出題 | 有害業務は範囲外 |
| 科目数 | 残り2科目×各10問 |

たとえば20問演習で14問正解（140点）でも、関係法令だけ35点なら不合格になるおそれがあります。免除後も3科目のうち受験する2科目で40点ラインを演習で確認してください。

4申込時の免除申請手順

科目免除は申込画面で申請し、要項指定の証明書類を添付します。受験資格とは別に免除要件を満たしているか確認してください。

| 手順 | 内容 |
| --- | --- |
| 1 | 受験資格表で申込可否を確認 |
| 2 | 免除表で対象行を特定 |
| 3 | 証明書類をスキャン·アップロード |
| 4 | 受験料8,800円を決済 |

  • 例えば10月11日試験·8月1日申込開始なら
  • 7月第4週までに資格証の写しを用意し
  • 申込チェックリストで写真·氏名表記も同時に確認し

免除不認可の場合の取り扱いは要項が正本です。

5免除なし受験者との学習差

科目免除がない場合は30問·おおむね3時間で全3科目を受験します。

1. 労働衛生（有害業務除く）·関係法令（有害除く）·労働生理
2. 各科目10問·各100点·40点足切り
3. 1問あたり約6分の時間配分練習
4. 免除対象か不明なら申込前に試験協会へ問い合わせ
5. 44問·100問は第一種（第二種は30問のみ）

  • たとえば6月開始·10月11日（日）13:30開始なら
  • 免除なしなら30問本番形式を2回
  • 免除ありなら20問·135分形式を2回実施し
  • 合格基準180点·各40点を演習で記録してください

6よくある質問

科目免除があると問題数はいくつですか？
1科目（10問）が免除されるため、20問になります。試験時間はおおむね2時間15分（135分）に短縮されます。たとえば10月11日（日）13:30開始なら、通常の3時間より早く終了するイメージです。ただし免除科目·対象資格は年度ごとに要項で確認してください。第二種は44問ではなく30問（免除時20問）です。
免除は申込後に追加できますか？
原則、申込時に免除申請·証明書類を提出します。例えば8月5日に通常申込を完了した後に免除対象と判明した場合、要項に定められた変更手続があるか試験協会へ確認してください。間に合わない場合は30問·3時間で受験する計画に切り替え、3科目40点足切りを意識した学習を続けるのが安全です。
免除者も受験料8,800円ですか？
第二種衛生管理者試験の受験料は、科目免除の有無にかかわらず8,800円（非課税）が目安です（年度で要項確認）。たとえば春期·秋期の2回受験する場合は合計17,600円になります。問題数が20問に減っても受験料が割引になるわけではありません。支払方法·期限は要項の「受験料」章が正本です。
