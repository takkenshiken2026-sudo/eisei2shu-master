第二種衛生管理者試験のリスクアセスメント手順【4ステップ·SDS·GHS連携】

第二種衛生管理者試験のリスクアセスメントは、化学物質による健康障害リスクを評価し低減措置を講じる4ステップ手順が労働衛生10問の定番論点です。SDS（安全データシート）·GHS（化学品の分類と表示）の情報をもとに実施し、3点関係の正本はSDS·GHS·リスクアセスメント記事です。優先順位付け·記録保存3年等の義務論点は優先順位·記録義務記事が正本、本記事は4ステップの手順·各段階の内容に特化します。有害業務（特定化学物質等）の深掘りは第二種出題外です。試験は30問·300点·180点（60%）·各科目40点以上·3科目×各10問·約3時間（10月11日（日）13:30開始）·受験料8,800円です。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、第7週の労働衛生10問で「手順の順序」1問誤答なら、本記事の4ステップ表を10分転記·7日後に同論点5問を解き直すのが定番です。正本は化学物質関連法令·公式テキストと安全衛生技術試験協会（https://www.exam.or.jp/）要項です。

この記事の要点

本記事を読むと、たとえば第7週の労働衛生10問演習前にリスクアセスメント4ステップ表を10分で転記し、「評価と低減措置の順序」誤答肢を見抜けるようになります。

1リスクアセスメント4ステップ手順表

第二種労働衛生10問では、4ステップの順序·各段階の内容が頻出です。手順の取り違え肢に注意します。

| ステップ | 内容 | 典型誤答肢 |
| --- | --- | --- |
| 1.把握 | 化学物質·用途の特定 | 評価から始める肢 |
| 2.評価 | ばく露·健康影響の評価 | 表示のみで終わる肢 |
| 3.低減 | リスク低減措置の検討 | 記録だけの肢 |
| 4.記録 | 実施内容の記録 | 記録不要の肢 |
| 情報源 | SDS·GHS | 代用できる肢は誤答 |

  • たとえば第7週の労働衛生10問で「リスクアセスメントは表示のみ」肢が出たら誤答
  • 評価·低減措置が正
  • と4ステップ表を声出しで確認し

優先順位·保存3年の詳細は優先順位·記録義務記事が正本です。有害業務の血中濃度管理等は第二種出題外です。

2karotein-sds-ghs記事との連携

リスクアセスメントは、SDS·GHSとセットで理解します。3点関係の正本はSDS·GHS記事です。

| 要素 | 役割 | 手順記事との接続 |
| --- | --- | --- |
| SDS | 16項目の安全情報 | ステップ1の情報源 |
| GHS | 分類·表示の統一 | ラベル·SDS補完 |
| リスクアセス | 評価·低減 | 4ステップ本体 |
| 優先順位 | 高リスク優先 | yusendo記事 |
| 記録 | 3年保存等 | yusendo記事 |

  • 例えば第8週演習前にSDS·GHS記事の3点関係表を10分転記し
  • 「GHS表示がSDSの代わりになる」肢を見抜いたうえで
  • 本記事の4ステップ順序を確認する流れが定番

7日後に化学物質2問を解き直してください。第一種100問の特定化学物質深掘りは第二種直前に外してください。

3risk-assessment-yusendo記事との役割分担

リスクアセスメントは2記事で分担します。本手順記事は4ステップ·各段階の内容が正本、優先順位·記録義務記事はリスクの優先度付け·記録保存3年·見直し義務が正本です。

| 論点 | 本手順記事 | 優先順位·記録記事 |
| --- | --- | --- |
| 4ステップ | 順序·内容 | — |
| 優先度 | 概要のみ | 高リスク優先の詳細 |
| 記録 | ステップ4の位置 | 3年保存·記載事項 |
| 見直し | — | 変更時の再実施 |
| 演習 | 手順順序問題 | 優先度·記録問題 |

  • たとえば第7週に本記事で4ステップを暗記し
  • 第8週に優先順位·記録記事で3年保存·優先度の表を転記
  • 第9週の労働衛生10問で手順1問·記録1問を正解
  • と順に学習する流れが定番

2記事を混同すると手順だけ暗記して記録義務を落としがちです。

4労働衛生10問の典型出題パターン

第二種では手順の順序·SDS情報の活用·低減措置の必要性が中心です。第一種レベルの専門管理は出題外です。

| パターン | 正答の型 | 誤答の型 |
| --- | --- | --- |
| 順序 | 把握→評価→低減→記録 | 評価省略 |
| SDS | 情報源として活用 | 不要·代用 |
| 低減 | 措置を講じる | 表示のみ |
| GHS | 分類·表示 | SDS代替 |
| 有害業務 | 第二種は基本論点のみ | 深掘り肢 |

  • 例えば第9週の労働衛生10問で化学物質2問中1問正解なら
  • 誤答1問が手順系か優先度系かを分類し
  • 手順系は本記事·優先度系はyusendo記事
  • と参照先を切り分け

化学物質毒性比較記事で3経路表も併用してください。

5演習連動の4ステップ暗記ドリル

4ステップは演習連動で定着させます。丸暗記より誤答肢との照合が効率的です。

1. 労働衛生10問演習で化学物質設問の誤答を★印
2. 誤答位置（順序·内容）を4ステップ表に追記（10分）
3. SDS·GHS記事の3点関係表と照合（5分）
4. 声出し10分×7日で4ステップ順序を確認
5. 7日後に同論点5問を解き直し（20分）

たとえば10月4日（日）13:30から30問演習で労働衛生48点·関係法令50点·労働生理44点·総合182点なら、化学物質論点は合格圏です。労働衛生頻出論点記事の8論点表と誤答位置を照合してください。受験料8,800円·各40点足切りは要項どおりです。

6よくある質問

4ステップの順番は暗記必須ですか？
はい。第二種労働衛生10問では「把握→評価→低減→記録」の順序を取り違える肢が繰り返し出ます。たとえば第7週演習前に4行表を紙に書き、1ステップずつ1語で役割を付ける10分ルーティンを2週続けると正答率が上がります。「評価を省略できる」肢は典型誤答です。優先順位·記録3年は優先順位·記録義務記事で確認してください。有害業務の管理濃度深掘りは第二種出題外です。7日後に★印問題を本番形式で解き直してください。
優先順位·記録義務記事との違いは？
本手順記事は4ステップの順序·各段階の内容が正本、優先順位·記録義務記事はリスクの優先度付け·記録保存3年·見直しタイミングが正本です。例えば第7週に本記事で4ステップ暗記、第8週にyusendo記事で優先度表·3年保存、第9週の労働衛生10問で手順1問·記録1問を正解、と順に読む流れが定番です。2記事を1つにまとめて読むと記録義務を落としがちです。SDS·GHS記事の3点関係表も先に確認してください。30問·各40点足切りは演習記録に毎回書いてください。
有害業務のリスクアセスメントは出ますか？
特定化学物質·有害業務の特殊管理深掘りは第二種30問の出題範囲外です。SDS·GHS·リスクアセスメントの基本4ステップは労働衛生10問の範囲です。たとえば第一種100問教材の特定化学物質章全15問を第二種直前に解いても、各40点足切りの演習記録は改善しにくいです。比較学習として線引きし、本記事の4ステップ表に集中してください。第一種·第二種の違い記事も併読できます。7日後に化学物質論点5問を解き直します。
