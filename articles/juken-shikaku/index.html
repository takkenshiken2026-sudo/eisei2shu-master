第二種衛生管理者試験の受験資格と実務経験の確認方法

第二種衛生管理者試験の受験資格は、学歴区分と労働衛生の実務年数の両方を満たす必要があります。たとえば10月11日（日）13:30開始の回を8月上旬に申し込むなら、6月第1週に要項の資格表で自分の行を特定し、7月第2週までに事業者証明書の記入依頼を済ませるのが定番です。本記事では申込前の確認手順をステップごとに表にまとめます。正本は安全衛生技術試験協会（https://www.exam.or.jp/）の受験資格表です。

この記事の要点

本記事を読むと、自分の学歴·実務年数が要項のどの行に該当するかと、たとえば高校卒·実務2年11ヶ月の場合にあと1ヶ月待つべきかが判断でき、無効申込を防げます。

  • 要項表で行を特定
  • 事業者証明書
  • 卒業証明書
  • 申込前に確定
  • 無効申込に注意

1要項の資格表で行を特定する

受験資格確認の第一歩は、要項の受験資格表で自分の行を1つ特定することです。学歴ごとに必要な労働衛生の実務年数が異なります。

| 学歴（代表例） | 実務年数（目安） | 添付書類 |
| --- | --- | --- |
| 大学·短大·高専卒 | 1年以上 | 卒業証明書·事業者証明書 |
| 高校·中等教育学校卒 | 3年以上 | 卒業証明書·事業者証明書 |
| 上記以外 | 10年以上 | 事業者証明書 |
| 判定 | 両方充足 | 要項のコード番号で確認 |

たとえば6月6日時点で短大2019年卒·実務2年なら、「短大卒·1年以上」行に該当するか確認します。10月11日試験まで残り18週なら6月第1週に区分確定が安全です。

2実務経験の数え方と確認

労働衛生の実務年数は、要項で定められた業務に該当する期間を数えます。自己申告だけでは足りず、事業者証明書で証明が必要です。

| 確認項目 | 方法 |
| --- | --- |
| 業務内容 | 要項の「実務」の定義と照合 |
| 期間 | 前職·現職の通算可否を確認 |
| 起算日 | 卒業年月·入社日との関係 |
| 証明 | 事業者証明書の記載 |

  • 例えば10月11日（日）試験を目標に
  • 前職2年·現職1年6ヶ月で合計3年6ヶ月なら
  • 高校卒区分（3年以上）を満たす可能性があり

ただし両方の勤務先で労働衛生の実務に該当する業務だったか、要項と照合し、不明点は申込前に試験協会へ問い合わせてください。

3事業者証明書の依頼と記入

事業者証明書は勤務先の事業者が記入·捺印します。申込締切（試験日2か月前）に間に合うよう早めに依頼します。

| ステップ | 時期（10月試験例） | 内容 |
| --- | --- | --- |
| 依頼 | 7月第2週 | 総務·人事へ記入依頼 |
| 記入 | 7月下旬 | 実務期間·業務内容 |
| 確認 | 8月第1週 | 氏名·期間の照合 |
| 申込 | 8月上旬 | 書類添付·オンライン申込 |

たとえば7月15日に依頼し、7月31日までに記入完了を目指します。

  • 実務9年11ヶ月の時点で申し込むと10年未満で無効になるおそれがあるため
  • 10年に達する月をカレンダーに印し
  • それ以降に申込んでください

4卒業証明書と氏名の一致確認

学歴区分を証明する卒業証明書は、申込画面の入力と完全一致させます。

| 項目 | チェック |
| --- | --- |
| 氏名 | 身分証·申込画面と一致 |
| 卒業年月 | 実務起算の参考 |
| 学校名 | 正式名称 |
| 旧字体 | 不一致なら再発行 |

例えば卒業証明書が「齊藤」·身分証が「斉藤」の場合、申込前に再発行または試験協会へ表記確認を行います。不一致のまま8,800円を支払うと受験無効になるリスクがあるため、7月中に書類をそろえておくのが定番です。

5確認完了後の申込と費用

資格が確定してから申込みます。第二種の試験形式も合わせて確認しておきましょう。

1. 受験資格：要項表の該当行を確定
2. 申込時期：希望試験日2か月前から先着受付
3. 受験料：8,800円（非課税）·資格不備は返金なし
4. 試験形式：30問·300点·各科目40点以上
5. 試験日時：10月11日（日）13:30開始·約3時間

  • たとえば6月に資格確定·10月11日試験なら
  • 8月上旬に申込·受験料8
  • 800円を支払い
  • 費用内訳記事でテキスト·交通費の予算も同時に確認し

44問·100問は第一種の数値であり、第二種では使いません。

6よくある質問

実務経験は前職も通算できますか？
労働衛生の実務に該当する業務であれば、前職と現職を合算できる場合があります。例えば前職2年·現職1年6ヶ月で合計3年6ヶ月なら、高校卒区分（3年以上）を満たします。ただし事業者証明書に記載できる期間·業務内容は要項どおりです。前職の証明が必要な場合は、退職前に記入依頼を済ませるか、要項の通算ルールを試験協会に確認してください。
資格不明のまま申し込むとどうなりますか？
受験資格を満たさない申込は無効となり、受験料8,800円の返金は原則ありません。たとえば実務9年11ヶ月の時点で申し込むと、10年未満で不合格扱いになるおそれがあります。申込前に要項表と自分の卒業証明書·勤務履歴を照合し、不明点は安全衛生技術試験協会へ問い合わせてから手続きしてください。
学歴がなくても受験できますか？
労働衛生の実務10年以上があれば、学歴を問わず受験できる区分があります。たとえば実務12年·中学卒の場合、事業者証明書のみで申込可能な行に該当します。大学卒1年·高校卒3年など、学歴がある場合は必要年数が短くなります。要項の受験資格表で自分の行を特定し、10年に1ヶ月足りない場合は翌月以降の試験回を選んでください。
