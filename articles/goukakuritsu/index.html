第二種衛生管理者の合格率・難易度【2024年度49.8%】年度別推移と合格のリアルを解説

第二種衛生管理者試験の合格率は、安全衛生技術試験協会の公表資料が正本です。2024年度（令和6年度）の合格率は49.8%と公表されていますが、年度·試験回で変動するため、必ずhttps://www.exam.or.jp/の最新資料で数字を確認してください。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、15問演習60％（90点換算）なら「49.8%より難しい」状態と判断し、60時間前後の学習計画を検討するのが定番です。合格率より30問·各科目40点足切りの演習得点を優先してください。

この記事の要点

本記事を読むと、たとえば2024年度49.8%を参考にしつつ、自分の15問演習60％なら60時間計画が必要、と合格率に惑わされず演習得点で難易度を測る方法が分かります。

  • 2024年度49.8%
  • 要項正本
  • 演習得点で測る
  • 30問3科目
  • 足切り40点

12024年度（令和6年度）49.8%の読み方

合格率49.8%は「受験者のうち合格した割合」であり、初学者の独学合格率と一致するとは限りません。数字の正本は安全衛生技術試験協会の公表資料です。

| 項目 | 内容 |
| --- | --- |
| 2024年度 | 49.8%（公表値·要項で再確認） |
| 意味 | 受験者ベースの合格割合 |
| 含む | 再受験者·現場経験者 |
| 正本 | exam.or.jp |

  • たとえば2024年度49.8%を見て「半分は余裕」と判断しても
  • 自分の最初の30問演習が150点（50%）·労働生理38点なら
  • 足切り不合格圏内

公表値は参考に、演習得点を優先してください。

年度別の合格率·受験者数は試験協会の資料で確認します。本サイトの数字は執筆時点の公表値であり、最新は要項が正本です。

  • 例えば2026年6月6日時点で合格率を調べるなら

  • 第二種衛生管理者の行を確認し

春期回·秋期回で分かれている年度もあるため、受けた回を特定してください。

3合格率だけで難易度を判断しない理由

第二種の難易度は、合格率より合格基準の厳しさが効きます。

| 要因 | 影響 |
| --- | --- |
| 足切り40点 | 1科目39点で不合格 |
| 30問 | 1問10点の比重が大きい |
| 再受験者 | 合格率を押し上げる |
| 現場経験 | 法令理解が早い層を含む |

  • たとえば合格率49.8%の年度でも
  • 初学者が15問演習で正答率60％（90点/150点換算）なら
  • 60時間前後の学習計画が現実的な目安

100問·44問は第一種の数値で、第二種は30問で評価してください。

4自分の演習得点で難易度を測る

他人の合格率より、自分の得点推移を2週間ごとに記録します。

| 演習結果 | 難易度の目安 |
| --- | --- |
| 30問180点以上·3科目40点 | 本番合格圏 |
| 30問150点·1科目38点 | 足切りリスク高 |
| 15問60％ | 60時間計画を検討 |
| 2週間で+10点 | 計画どおり |

例えば6月の15問演習60％→8月の30問演習186点（労働生理42点）なら、難易度は「自分にとって平均的」に近づいています。40点未満科目がある場合は、その科目に週3時間を追加してください。

5再受験と合格のリアル

合格率に再受験者が含まれるため、初回受験の体感とはズレます。

1. 不合格の典型：1科目38点以下の足切り
2. 再受験：8,800円·2か月前先着申込
3. 独学初学者：60時間前後·週4〜6時間が目安
4. 3科目バランス：得意科目偏重が落ちる原因
5. 正本確認：合格基準·合格率は要項·公表資料

  • たとえば10月11日（日）13:30開始の初回受験で総合177点·労働生理41点なら
  • 総合点あと3点·足切りはクリア
  • 次回回に向け誤答解き直し中心の4週計画が定番

6よくある質問

2024年度の合格率49.8%は信頼できますか？
安全衛生技術試験協会の公表資料に基づく値ですが、執筆後に訂正·更新がある場合があります。たとえば2026年6月に調べるなら、https://www.exam.or.jp/の試験結果ページで第二種衛生管理者の最新行を開き、年度·合格率を再確認してください。本記事の数字は参考値として扱い、要項正本を優先します。
合格率50％なら独学で半分は合格できますか？
合格率は再受験者を含むため、初学者の独学合格率とは一致しません。目安は60時間前後の学習で、週4〜6時間×3か月の確保が現実的です。例えば6月から10月11日試験まで18週×4時間＝72時間なら、初学者目安の範囲内です。3科目40点足切りを演習で常に確認してください。
30問だから合格率より簡単ですか？
問題数が少ないからといって簡単とは限りません。30問·300点·各科目40点足切り·おおむね3時間という条件があり、初学者が最初の演習で150点（50%）なら「合格率49.8%より難しい」状態です。たとえば総合195点でも労働生理38点なら不合格です。第一種の100問基準ではなく、第二種の30問基準で評価してください。
