第二種衛生管理者試験の合格率

第二種衛生管理者試験の合格率は、安全衛生技術試験協会が公表する統計です。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、2024年度（令和6年度）の公表値49.8%を参考にしつつ、必ずhttps://www.exam.or.jp/の最新資料で再確認するのが定番です。本記事では合格率の確認先と、数字の基本的な意味を表にまとめます。合格点の内訳や足切り40点は合格基準記事、統計の深い読み方は別記事を参照してください。

この記事の要点

本記事を読むと、合格率をどこで確認し、2024年度49.8%のような公表値をどう扱えばよいかが分かり、非公式まとめサイトの数字と混同するリスクを減らせます。

  • exam.or.jpが正本
  • 2024年度49.8%例
  • 年度で変動
  • 合格率≠合格点
  • 演習得点を優先

1合格率の確認先（正本）

第二種衛生管理者試験の合格率は、安全衛生技術試験協会の公表資料が正本です。ブログやSNSのまとめは年度·出典が不明なことが多く、学習計画の根拠に使わないでください。

| 確認先 | 内容 |
| --- | --- |
| https://www.exam.or.jp/ | 試験結果·統計 |
| 要項·試験案内 | 試験回ごとの日程 |
| 公表資料 | 受験者数·合格者数·合格率 |

  • たとえば2026年6月6日時点で調べるなら
  • 試験結果ページで「第二種衛生管理者」の行を開き
  • 最新年度の合格率·受験者数をその場で控え

執筆時点の数字は参考値であり、閲覧時に必ず再確認してください。

22024年度49.8%の読み方（基本）

合格率49.8%は「その年度の受験者のうち合格した割合」を示します。初学者が「半分は合格できる」と単純換算する前に、次の前提を押さえます。

| 項目 | 意味 |
| --- | --- |
| 49.8% | 公表資料の参考値（要再確認） |
| 母数 | 受験者全体（再受験者含む） |
| 合格条件 | 総合180点·各科目40点以上 |
| 試験形式 | 30問·300点·約3時間 |

例えば2024年度49.8%を見た場合、「自分も50%の確率で合格」とは限りません。再受験者の比率や学習時間は統計に含まれないため、統計の深い読み方は合格率·統計の読み方記事に進んでください。

3合格率と合格点は別の話

合格率は「結果の割合」、合格点は「個人の得点基準」です。

  • 混同すると
  • たとえば演習150点（50%）を「合格率並み」と誤解し
  • 足切り40点の確認を怠ることがあり

| 概念 | 第二種の数値（例） |
| --- | --- |
| 合格率 | 年度ごとに公表（例49.8%） |
| 合格点（総合） | 180点以上（300点中） |
| 合格点（科目） | 各40点以上（100点中） |
| 問題数 | 30問（3科目×各10問） |

  • たとえば10月11日（日）13:30開始の回を目標に15問演習で90点（60%）なら
  • 合格率49.8%より高い割合ですが
  • 科目別40点未満があれば本番不合格

合格点の計算方法は合格点記事、足切りの詳細は合格基準記事を参照してください。

4年度·試験回で変わる理由

合格率は年度·春期回·秋期回で変動します。受験者数の増減·再受験者の比率·出題難易度の差が影響するため、1年の数字だけで「簡単になった」と判断するのは危険です。

| 変動要因 | 例 |
| --- | --- |
| 受験者数 | 増加で合格率が下がる年度も |
| 再受験者 | 学習済み層が含まれる |
| 試験回 | 春期·秋期で母数が異なる |
| 出題 | 科目バランスの年度差 |

例えば2023年度50.2%·2024年度49.8%のように1%前後で動いても、個人の演習得点が180点·3科目40点に届いているかの方が重要です。年度推移の読み方は統計の読み方記事で詳述します。

5合格率を学習計画に活かすコツ

合格率は「目標設定の参考」に留め、合格判断は自分の演習得点で行います。

1. 公表合格率を確認したら、すぐ30問演習の得点記録に切り替える
2. 3科目40点足切りを演習のたびにチェックする
3. 44問·100問は第一種の数値（第二種は30問）と混同しない
4. 有害業務は第二種出題外（比較学習のみ）
5. 初学者は60時間前後の学習目安をカレンダーに書く

  • たとえば6月6日時点で10月11日（日）13:30開始まで残り18週なら
  • 第1週末の15問演習得点と科目別点数を記録し
  • 合格率49.8%より「自分にとっての難易度」を測るのが定番

難易度の総合判断は合格率·難易度記事と組み合わせてください。

6よくある質問

第二種の合格率はどこで確認できますか？
安全衛生技術試験協会の公式サイト（https://www.exam.or.jp/）の試験結果·統計ページが正本です。たとえば2026年6月に調べるなら、「第二種衛生管理者」の行を開き、最新年度の合格率·受験者数·合格者数を確認してください。ブログのまとめは出典·年度が不明なことが多いため、学習計画の根拠には使わないでください。
2024年度の合格率49.8%は信頼できますか？
安全衛生技術試験協会の公表資料に基づく値ですが、執筆後に訂正·更新がある場合があります。たとえば2026年6月6日時点で調べるなら、exam.or.jpで第二種衛生管理者の最新行を開き、年度·合格率をその場で再確認してください。本記事の数字は参考値として扱い、要項正本を優先します。
合格率50％なら半分は合格できますか？
合格率は再受験者を含む全体の割合であり、初学者の独学合格率とは一致しません。たとえば2024年度49.8%でも、最初の15問演習で60点（40%）なら「合格率より難しい」状態です。目安は60時間前後の学習で、3科目40点足切りを演習で常に確認してください。
