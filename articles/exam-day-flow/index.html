2受付・着席前に確認すること

受付では受験票と要項で求められる身分証を提示します。氏名·試験日·会場名が申込内容と一致しているか、その場で確認してください。

| 確認項目 | ポイント |
| --- | --- |
| 受験票 | 氏名·日付·会場·13:30 |
| 身分証 | 有効期限·写真付き |
| 筆記用具 | HB鉛筆·消しゴム |
| 座席 | 受験票の座席番号 |

  • 例えば受験票の開始時刻が13:30で会場が東京センターなら
  • 前日に持ち物を袋に入れ
  • 当日朝は会場案内の入口から入り

受付で不備があれば早めに係員へ声をかけてください。

3試験本番（30問·約3時間）の進め方

試験開始後は、30問·300点満点·5肢択一マークシートに解答します。出題は3科目（労働衛生·関係法令·労働生理）×各10問で、有害業務は出題範囲外です。

| 項目 | 内容 |
| --- | --- |
| 問題数 | 30問（各科目10問） |
| 形式 | 5肢択一マークシート |
| 時間 | おおむね3時間 |
| 配分目安 | 約6分/問 |

たとえば第1〜10問（労働衛生）に60分·第11〜20問（関係法令）に60分·第21〜30問（労働生理）に50分·見直し10分、という配分で進める受験者が多いです。時間配分記事と合わせて本番前に30問演習を1回実施してください。

4答案提出と退場

試験終了時刻（または終了合図）に従い、答案用紙を提出します。未記入欄がないか最終10分で確認し、提出後は指示があるまで席を離れないでください。

| 段階 | 注意 |
| --- | --- |
| 終了5分前 | 未記入欄チェック |
| 提出 | 答案用紙を係員へ |
| 退場 | 会場指示に従う |

  • 例えば16:25頃に「あと5分」のアナウンスがあれば
  • 30問すべてにマークがあるか確認し
  • 消しゴムのカスを落とさないよう注意し

提出完了を係員に確認してから退場してください。

6よくある質問

試験は何時から始まりますか？
おおむね13:30開始·約3時間です。たとえば2026年10月11日（日）13:30開始の回なら、16:30頃終了のイメージですが、終了時刻は会場案内に従ってください。30問·5肢択一マークシートで、3科目×各10問が出題されます。開始30〜45分前の到着を目安にしてください。
受付には何が必要ですか？
受験票と要項で求められる身分証が基本です。例えば運転免許証と受験票をセットで提示し、氏名·試験日·会場名が申込内容と一致するか確認します。筆記用具（HB鉛筆·消しゴム）は着席後に机の上に準備してください。持ち物記事で前日チェックリストを確認してください。
試験中にトイレに行けますか？
要項·会場の指示に従います。たとえば試験開始前の待機時間に済ませ、本番中は係員の指示がある場合のみ退席、という運用が一般的です。10月11日（日）13:30開始なら12:45到着後に済ませておくと安心です。体調不良の場合は早めに係員へ相談してください。
