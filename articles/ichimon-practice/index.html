第二種衛生管理者試験の一問一答の活用法

第二種衛生管理者試験は30問·300点·3科目×各10問·総合180点（60%）以上かつ各科目40点以上が合格条件です。一問一答は、数値·用語·主体を短時間で定着させる道具で、30問本番形式の演習と役割が異なります。二衛マスターには657問の一問一答と210問の実践演習があり、有害業務の深掘りは第一種向けで第二種出題外です。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、平日30分×一問一答10問·日曜30問本番形式、の2段構成が定番です。本記事では一問一答の使い方と30問演習への切替タイミングを表に整理します。正本は安全衛生技術試験協会（https://www.exam.or.jp/）要項です。

この記事の要点

本記事を読むと、たとえば関係法令の一問一答20問で選任人数·保存年数を確認した後、日曜30問演習で40点足切りを記録する流れが設計できます。

  • 657問一問一答
  • 30問本番形式
  • 誤答4分類
  • 各40点足切り
  • 有害業務は出題外

1一問一答と30問本番形式の役割分担

一問一答は暗記定着、30問演習は時間配分と足切り確認に使い分けます。第二種は1問10点·5肢択一·おおむね3時間（13:30開始）です。

| 演習形式 | 向く内容 | 目安時間 |
| --- | --- | --- |
| 一問一答 | 数値·用語·主体 | 1問1〜2分 |
| 科目別10問 | 分野の弱点特定 | 30〜40分 |
| 30問本番形式 | 180点·各40点確認 | 約180分 |
| 第一種100問 | 出題外の範囲 | 第二種では不使用 |

  • たとえば平日22:00〜22:30に一問一答10問
  • 日曜13:30から30問演習2時間
  • とカレンダーに書くと657問を3か月で消化しつつ本番ペースも体験でき

1問1答演習のやり方記事で解き方の詳細を確認してください。

2科目別10問×3周の進め方

一問一答は3科目（労働衛生·関係法令·労働生理）×各10問ずつ周回する方式が定着しやすいです。

| 周 | 重点 | 目標 |
| --- | --- | --- |
| 第1周 | 全肢を読む | 正答率50%以上 |
| 第2周 | 誤答のみ | 同論点で正解 |
| 第3周 | 混同語 | 3科目40点以上 |

  • 例えば第2週の関係法令10問で6問正解（60点）·労働生理10問で3問正解（30点）なら
  • 第3週は労働生理の一問一答15問を優先し
  • 関係法令は週5問に減らし

演習問題量記事で週次の解く量の目安も確認してください。

3誤答の4分類と解き直しサイクル

一問一答の誤答は次の4分類で記録すると、30問演習での再発を防げます。

| 分類 | 例 | 対策 |
| --- | --- | --- |
| 数値 | 50人·14日 | 数値リスト |
| 用語 | 急性·慢性 | 用語解説 |
| 主体 | 事業者·医師 | 比較表 |
| 計算 | WBGT換算 | 計算記事 |

  • たとえば一問一答10問中4問誤答·うち3問が「50人選任」の数値混同なら
  • 24時間後に同10問を解き直し
  • 7日後に★印5問だけ再挑戦するサイクルをブックマーク復習記事どおりカレンダー登録し

4657問一問一答の優先順位付け

657問すべてを均等に解く必要はありません。演習得点から優先科目を決めます。

| 優先 | 条件 | 一問一答量 |
| --- | --- | --- |
| ★★ | 40点未満が2回 | 週20問 |
| ★ | 40〜49点 | 週10問 |
| | 50点以上 | 週5問維持 |
| 全体 | 直前4週 | 新規より解き直し |

  • 例えば第8週の30問演習で労働生理38点·関係法令55点·労働衛生52点なら
  • 労働生理の一問一答を週20問に増やし
  • 657問のうち労働生理タグ付きから優先的に解き

用語ハブ活用法記事で誤答語を用語解説へリンクしてください。

530問演習へ切り替えるタイミング

一問一答中心から30問本番形式へ移る目安は次のとおりです。

1. 科目別10問で3科目すべて40点以上が2週連続
2. 一問一答の解き直し★印が20問以下
3. 残り8週以上ある（30問演習を4回以上入れる余裕）
4. 13:30開始·約3時間の時間感覚を1回体験済み
5. 44問·100問用の第一種教材を手元から外す

  • たとえば第6週·第7週の科目別10問で関係法令42点·労働衛生48点·労働生理41点が2週続いたなら
  • 第8週日曜から30問本番形式に切替え
  • 一問一答は誤答★印のみ解き直す運用にし

過去問の使い方記事で30問演習の記録表を確認してください。

6よくある質問

一問一答だけで合格できますか？
一問一答だけでは足りません。657問で用語·数値は定着しますが、30問·約3時間の時間配分と各科目40点の同時達成は本番形式演習でしか確認できません。たとえば一問一答正答率80%でも30問演習150点·労働生理35点なら不合格です。科目別10問→30問本番形式の順で段階を上げてください。
657問は全部解く必要がありますか？
全部解く必要はありません。40点未満科目のタグ付き一問一答を優先し、50点以上の科目は解き直し中心で十分です。例えば残り10週·週15問なら150問消化でき、657問のうち弱点分野200問に絞る方が得点効率が高いです。演習問題量記事で週次の解く量の上限も確認してください。
第一種の一問一答を使ってもよいですか？
第二種向けを優先してください。第一種100問·有害業務·特定化学物質の深掘りは第二種の出題範囲外です。たとえば44問60%合格の第一種問題集を第二種直前に解くと、30問·180点·各40点の合格基準と時間配分がずれます。第一種·第二種の違い記事で出題科目を再確認してください。
