第二種衛生管理者試験のノートの取り方【3科目·657語·30問演習向け】

第二種衛生管理者試験のノートは、3科目×各10問·657語用語解説·30問演習記録を1冊（またはアプリ）に整理する道具です。科目別得点·誤答4分類·★印7日解き直しを演習のたびに転記すると、各科目40点足切りの底上げが速くなります。誤答の詳細フォーマットは間違いノート記事が正本で、本記事はノート構成·科目別レイアウトに特化します。試験は30問·300点·180点（60%）·各科目40点以上·有害業務除外·約3時間（10月11日（日）13:30開始）·受験料8,800円です。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、第1週に3科目分のノート見開き構成を決め、第2週の日曜30問演習後20分で得点表を転記するのが定番です。正本は安全衛生技術試験協会（https://www.exam.or.jp/）要項です。

この記事の要点

本記事を読むと、たとえば第2週の30問演習後に関係法令52点·労働生理39点·誤答12問を3科目ノートに20分で転記し、40点未満科目の★リストが作れます。

  • 3科目構成
  • 657語連動
  • 30問演習記録
  • ★印7日解き直し
  • 各科目40点

13科目×10問向けノート構成（見開き3分割）

ノートは科目別にページを分けると、40点未満科目が一目で分かります。1演習1ページが振り返りの基本です。

| ページ | 内容 | 記入例 |
| --- | --- | --- |
| 関係法令 | 10問得点·誤答 | 52点·Q3,Q7 |
| 労働衛生 | 10問得点·誤答 | 48点·Q12 |
| 労働生理 | 10問得点·誤答 | 39点·Q18,Q22 |
| 総合 | 30問·180点確認 | 179点·足切り注意 |
| ★ | 7日後解き直し | Q18,Q22 |

  • たとえば第3週日曜30問演習後20分で上表を転記し
  • 労働生理39点ページに★2問を書き込み
  • 7日後の日曜に★2問本番形式で解き直し

44問·100問の第一種演習記録は第二種ノートに混ぜないでください。

2657語用語解説との連動（誤答語1行メモ）

657語用語解説は定義の正本です。ノートには演習誤答語だけ1行メモで十分です。

| ステップ | 行動 | 時間 |
| --- | --- | --- |
| 1 | 演習誤答語を拾う | 5分 |
| 2 | 用語解説で定義確認 | 10分 |
| 3 | 科目ページに1行メモ | 3分 |
| 4 | 2語以上混同なら比較表 | 10分 |
| 5 | 7日後同論点10問 | 20分 |

  • 例えば第4週の関係法令10問で「衛生管理者·産業医·事業者」を混同したら
  • 用語解説で各定義を確認し
  • 関係法令ページに主体3語を1行で書き
  • 用語ハブ活用法記事の4ハブ役割分担表と突合し

657語すべて暗記は不要で、誤答から広げます。

330問演習記録1枚フォーマット（転記20分）

30問演習のたびに同じフォーマットで転記すると、18週分の得点推移が比較しやすくなります。

| 欄 | 記入例 |
| --- | --- |
| 日付·演習 | 6/8日曜30問 |
| 得点 | 関52·衛48·生39·総179 |
| 正解数 | 19問/30問 |
| 誤答番号 | Q3,Q7,Q18… |
| ★リスト | Q18,Q22（7日後） |

たとえば6月8日（日）13:30から30問演習で総合19問正解（190点）·労働生理3問正解（30点）なら、総合合格圏でも足切り不合格圏、と赤字でノートに書き込みます。合格基準記事で180点·各40点の両条件も確認してください。一問一答657問は平日30分·日曜30問の2段構成が定番です。

4間違いノート記事との役割分担

ノート運用は2記事に分けます。間違いノート記事は4分類·★印の詳細、本ノート記事は3科目構成·演習記録が正本です。

| 論点 | 間違いノート記事 | 本ノート記事 |
| --- | --- | --- |
| 4分類 | 数値·主体·条文·読飛 | — |
| 科目構成 | — | 3分割レイアウト |
| ★解き直し | 7日サイクル詳細 | リスト欄 |
| 657語 | — | 1行メモ連動 |
| 30問記録 | 1演習1ページ | フォーマット |

  • 例えば日曜30問演習後に間違いノート記事どおり4分類転記（10分）
  • 本記事どおり科目別得点表更新（10分）
  • と合計20分で週次ノートが完成し

ブックマーク復習記事の電子★とも併用できます。

5直前2週間のノート運用（★中心）

試験直前は新規インプットより★解き直しとノート見返しが得点効率が高いです。

1. ★リストを科目別に並べ替え（10分）
2. 40点未満科目の★を全問解き直し
3. 657語メモのうち★関連語だけ声出し
4. 新規30問は週1回→隔週に減らす
5. 10月4日（日）最終30問·10月11日は★のみ

  • たとえば10月11日（日）13:30開始の2週前·9月27日（日）に最終30問演習し
  • 10月4日（日）は★印15問解き直しのみ
  • と直前カレンダーを固定し

演習問題量記事で週次30問回数の目安も確認してください。有害業務の深掘り誤答は第二種出題外として★から外します。

6よくある質問

ノートは1冊と科目別、どちらがよいですか？
1冊内で科目別にページを分ける見開き3分割が定番です。別冊にすると得点推移の比較がしづらくなります。たとえば第2週にA4ノート1冊の最初3見開きを関係法令·労働衛生·労働生理に割り当て、日曜30問演習後20分で得点表を転記するのが効率的です。40点未満科目のページを週次で厚くしてください。間違いノート記事の4分類表と併用すると★管理が楽になります。44問演習の記録は第二種ノートに載せないでください。
657語すべてノートに書く必要がありますか？
必要ありません。演習の誤答語·比較表の頻出組·一問一答657問の弱点タグから必要語だけ1行メモで広げるのが効率的です。例えば9月第2週時点で混同リストが12語なら、新規20語追加より既存12語の7日後解き直し2周の方が30問の関係法令40点足切りに効きます。用語ハブ活用法記事の4ハブ役割分担表も確認してください。残り4週未満なら新語追加を止め★解き直しに切り替えます。
間違いノート記事との違いは？
間違いノート記事は4分類·★印·7日解き直しの詳細が正本、本ノート記事は3科目構成·30問演習記録フォーマットが正本です。たとえば日曜30問演習後に間違いノート記事どおり4分類10分、本記事どおり科目別得点表10分、と合計20分で週次ノートが完成します。電子ブックマークのみだと分類理由が薄くなるため、1行メモは紙ノートに残してください。7日後に★問題を本番形式で解き直します。
