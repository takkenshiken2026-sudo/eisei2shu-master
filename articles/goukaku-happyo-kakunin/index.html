第二種衛生管理者の合格発表はいつ？確認方法・免許申請の流れまで解説

第二種衛生管理者試験の合格発表日·合否確認方法·免許申請の流れは、年度·試験回ごとに安全衛生技術試験協会の要項·試験案内が正本です。たとえば10月11日（日）13:30開始の秋期回を受けた場合、合格発表は試験日からおおむね2週間後（要項で日時確認）にWeb等で確認し、合格後は免許申請費1,500円程度（要項確認）の手続が必要です。本記事では発表から免許証到着までの流れを表にまとめます。

この記事の要点

本記事を読むと、たとえば10月11日試験後に合否確認URL·受験番号·免許申請の必要書類をいつ何を用意すべきかが分かり、合格後の手続き漏れを防げます。

  • 要項が正本
  • 合否Web確認
  • 免許申請1,500円程度
  • 受験番号控え
  • 30問·180点合格

1合格発表の時期（試験回ごと）

合格発表日は試験日·試験回ごとに異なります。必ず安全衛生技術試験協会の試験案内·要項で最新日時を確認してください。

| 試験回（例） | 試験日（例） | 発表目安 |
| --- | --- | --- |
| 春期回 | 4月の日曜 | 要項記載 |
| 秋期回 | 10月11日（日） | 試験後おおむね2週間 |
| 確認 | exam.or.jp | 正本 |

  • たとえば2026年10月11日（日）13:30開始の回なら
  • 要項の「合格発表」欄で10月下旬の日時·時刻（10:00等）を確認し
  • カレンダーに登録し

44問·100問は第一種の形式で、第二種は30問·300点です。

2合否確認の方法（Web·郵送）

近年はWebでの合否確認が中心ですが、年度·試験回で手順が異なります。

| 項目 | 確認 |
| --- | --- |
| 確認手段 | 要項（Web·郵送等） |
| 必要情報 | 受験番号·生年月日等 |
| 発表時刻 | 要項記載（混雑注意） |
| 正本 | https://www.exam.or.jp/ |

例えば発表日10:00開始の年度で。

  • 9:55にログインし
  • 受験番号123456·生年月日を入力して確認し

サイト混雑でエラーが出る場合があるため、午前中以外の時間帯も要項で確認できるか調べておいてください。

3合格通知·合格証明の取り扱い

合格後に届く書類·データの保管方法です。要項·案内が正本です。

| 書類 | 用途 |
| --- | --- |
| 合格通知 | 免許申請の添付 |
| 合格証明 | 会社提出·保管 |
| 控え | 紛失時の問い合わせ |
| 保管 | 試験協会指示どおり |

  • たとえば10月下旬に合格通知が届いたら
  • 11月第1週までに免許申請書類と一緒にコピーを取り
  • 原本は封筒に入れて保管し

合格基準（総合180点·各科目40点以上）を満たした結果であることを通知書の記載と照合してください。

4免許申請の流れと費用

第二種衛生管理者の免許は、試験合格後に別途申請が必要です。

| 手順 | 内容 |
| --- | --- |
| 1 | 要項で申請書式·期限を確認 |
| 2 | 合格通知等を添付 |
| 3 | 手数料·収入印紙1,500円程度 |
| 4 | 指定先へ郵送·提出 |

例えば合格通知到着から2週間以内に申請期限がある年度で。

  • 到着3日以内に印紙1
  • 500円·写真·申請書をそろえ
  • 追跡可能な郵送で提出し

金額·書類は年度で変わるため、免許申請記事と要項を必ず照合してください。

5免許証到着までの目安と注意点

免許証が届くまでの期間も要項·案内が正本です。

1. 申請から1〜2か月で免許証到着（年度で変動·要項確認）
2. 申請漏れ·書類不備があると到着が遅れる
3. 届くまで合格通知の控えを保管する
4. 会社への報告は免許証到着前でも合格通知で可能な場合あり
5. 再受験は8,800円·別日程（不合格時）

  • たとえば10月11日試験·10月下旬合格·11月上旬申請なら
  • 12〜1月に免許証到着を見込み
  • 届くまで申請控えを保管し

公式情報の確認先記事でexam.or.jpの更新を定期的に確認してください。

6よくある質問

合格発表はいつ頃ですか？
試験日·試験回ごとに異なり、安全衛生技術試験協会の要項が正本です。たとえば10月11日（日）13:30開始の秋期回なら、試験後おおむね2週間後（10月下旬）にWeb等で発表されることが多いですが、必ず最新要項の「合格発表」欄で日時·時刻を確認してください。受験票·申込控えに受験番号を記録しておくと当日スムーズです。
合格後すぐに現場で衛生管理者になれますか？
試験合格と免許交付は別です。合格通知後、要項どおりの免許申請·手数料1,500円程度（要項確認）を経て免許証が交付されます。例えば11月1日に申請·12月15日に免許証到着の場合、交付日以降に選任要件を満たす形で衛生管理者として就く流れが典型です。会社の総務·人事と申請期限を早めに共有してください。
不合格の場合はどう確認しますか？
合否確認の方法は合格時と同じです。第二種は総合180点未満、または1科目でも40点未満（足切り）で不合格になります。たとえば総合195点·労働生理38点なら不合格です。再受験は8,800円·希望試験日2か月前先着申込が必要です。合格基準記事で科目別得点を確認し、次回に向け40点未満科目を重点復習してください。
