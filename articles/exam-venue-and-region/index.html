第二種衛生管理者試験の受験地・会場の確認

第二種衛生管理者試験の受験地は、申込時に試験センター単位で先着順に確保します。たとえば10月11日（日）13:30開始の回を受けるなら、8月上旬の受付開始日に第1希望都市の定員を確保し、受験票到着後に会場名·開始時刻を申込内容と照合するのが定番です。本記事では受験地選択から当日までの会場確認を表にまとめます。正本は安全衛生技術試験協会（https://www.exam.or.jp/）の最新要項です。

この記事の要点

本記事を読むと、たとえば関東在住で第1希望が満席だった場合の第2希望センター選びや、受験票の会場名とGoogleマップの住所照合が分かり、試験当日の迷子·遅刻リスクを下げられます。

  • 先着定員
  • 受験票で会場確認
  • 13:30開始
  • 2か月前申込
  • 要項が正本

1試験センターと受験地の選び方

第二種衛生管理者試験は、全国の試験センターで実施されます。受験地は「都道府県·都市」ではなくセンター単位で定員があり、先着順で締切になります。

| 選択の観点 | 例 |
| --- | --- |
| 通勤距離 | 当日60分以内が目安 |
| 定員 | 受付開始日に確保 |
| 会場規模 | 要項の実施場所一覧 |
| 併願 | 同日13:30の他試験と衝突確認 |

  • たとえば10月11日（日）13:30開始の回を関東で受ける場合
  • 8月1日の受付開始日にログインし
  • 第1希望が満席なら当日中に第2希望センターまで確定し

試験日程記事と併願ルール記事を並べて確認してください。

2申込時の受験地確保手順

受験地は希望試験日の2か月前から先着順で申込みます。第二種の受験料は8,800円（非課税）で、受験地確保は申込画面の一部です。

| 手順 | 内容 |
| --- | --- |
| 1 | 要項で受験資格を確認 |
| 2 | 希望試験日·センターを選択 |
| 3 | 受験料8,800円を決済 |
| 4 | 完了メール·控えを保存 |

例えば10月11日試験なら8月上旬が受付開始目安です。

残り18週の6月6日時点で。

  • 8月1日をカレンダー登録し
  • 申込チェックリストで写真·資格·決済を先にそろえておくと受験地選択に集中でき

3受験票到着後の会場確認

申込完了後、受験票（または受験票データ）が届きます。会場名·試験日·開始時刻13:30·氏名が申込内容と一致するか必ず照合します。

| 確認項目 | 照合先 |
| --- | --- |
| 会場名·住所 | 要項·公式の実施場所 |
| 試験日·時刻 | 13:30開始·約3時間 |
| 氏名 | 身分証·申込画面 |
| 試験種別 | 第二種（30問） |

たとえば受験票に「○○試験センター3号館」と記載されたら、公式サイトの会場案内PDFと建物名·入口を照合します。不一致がある場合は、試験協会へ申込期間内に問い合わせてください。

4会場変更と当日の交通

受験地の変更は要項に定められた期間·手続に従います。定員に空きがあれば変更できる場合もありますが、締切直前は満席になりやすいため早めの手続が安全です。

| タイミング | 確認 |
| --- | --- |
| 申込後 | 変更可否·手数料 |
| 1か月前 | 会場案内·集合時間 |
| 前日 | 交通ルート·所要時間 |
| 当日 | 12:45頃到着目安 |

例えば10月11日（日）13:30開始で会場まで電車45分なら、11:30出発·12:15到着·12:45受付完了を目標にします。当日の流れ記事と合わせ、遅延に備えて前日にルートをGoogleマップで確認してください。

5受験地まわりで起きやすい不備

受験地·会場に関する典型的な不備は次のとおりです。

1. 定員満了で希望都市を受験できない（申込が遅い）
2. 受験票の会場名と当日の案内板が異なる（別館·改修）
3. 13:30開始なのに12:00到着と勘違いし、長時間待機で集中力が落ちる
4. 第一種と第二種の会場を混同（同日13:30は併受験不可）
5. 会場変更手続を締切後に行い、変更できない

  • たとえば8月5日時点で第1希望が満席なら
  • その日のうちに第2希望を確定し
  • 受験票到着後に再度会場名を確認し

試験は30問·約3時間·5肢択一です。44問·100問は第一種の数値です。

6よくある質問

受験地はいつ決めればよいですか？
希望試験日の2か月前から先着順で申込み、受験地も同時に確保します。たとえば10月11日（日）13:30開始の回なら8月上旬の受付開始日にログインし、第1希望センターを早めに選ぶのが定番です。定員満了になると別センター·別都市を選ぶ必要があるため、受付開始から1週間以内の申込がおすすめです。
受験票の会場名が分からない場合は？
受験票に記載された会場名·住所を、安全衛生技術試験協会の試験実施場所ページ（https://www.exam.or.jp/h_aramashi/）と照合してください。例えば「○○センター」とだけ書かれている場合、公式の会場案内PDFで建物·入口·最寄り駅を確認します。試験1週間前になっても不明なら、試験協会へ問い合わせてください。
受験地を変更できますか？
要項に定められた期間·手続があれば変更できる場合があります。たとえば9月15日時点で転勤が決まり会場が遠くなった場合、要項の「受験地変更」を確認し、空きがあるセンターへ申請します。締切直前は満席になりやすいため、変更が必要なら早めに手続してください。変更不可の場合は交通·宿泊計画を前日までに確定します。
