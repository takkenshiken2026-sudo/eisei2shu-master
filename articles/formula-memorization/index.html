第二種衛生管理者試験の数式·数値の暗記法

第二種衛生管理者試験では、数式そのものより数値·単位·期限の「一語差」が30問·3科目×各10問で頻出します。合格は総合180点（60%）以上·各科目40点以上·有害業務除外·約3時間（10月11日（日）13:30開始）です。44問·100問の第一種専門数値は第二種ノートに載せないでください。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、第1週に数値·期限早見表を転記し、第2週の関係法令10問で保存年数2問·第3週の労働衛生10問でWBGT2問の正答率を記録する、の演習連動暗記が定番です。正本は安全衛生技術試験協会（https://www.exam.or.jp/）要項と公式テキスト·用語解説です。

この記事の要点

本記事を読むと、たとえば「3年と5年の保存年数」「18%と19%の酸素濃度」を暗記カードではなく演習誤答から拾い、数値早見表→7日後解き直しのサイクルで定着できます。

  • 演習連動暗記
  • 数値·期限早見表
  • 一語差に注意
  • 7日後解き直し
  • 44問数値は除外

1第二種で覚える数値の種類

数式·数値暗記は科目横断で整理します。全部を一度に覚える必要はありません。

| 種類 | 例 | 主な科目 |
| --- | --- | --- |
| 人数基準 | 常時50人 | 関係法令10問 |
| 保存年数 | 3年·5年 | 関係法令10問 |
| 濃度·基準 | 18%·10ppm | 関係法令·労働衛生 |
| 温熱 | WBGT·休息 | 労働衛生·労働生理 |
| 換気·風量 | m³/h·m² | 労働衛生10問 |

  • たとえば8月第1週の関係法令10問で「議事録3年」と「健康診断結果5年」を混同したら
  • 数値·期限早見表の保存年数欄に2行追加し
  • 翌週同論点10問で2問正解を目標にし

第一種44問教材の有害業務専門濃度は第二種出題外のため暗記帳から除外してください。

2数値·期限早見表の作り方

暗記カードより、演習誤答から広げる早見表の方が定着しやすいです。

| 列 | 記入例 | 更新タイミング |
| --- | --- | --- |
| 数値 | 18%·50人 | 誤答のたび |
| 制度 | 衛生委員会 | 比較表とリンク |
| 科目 | 関係法令 | 演習後 |
| 正誤 | 7/10正解 | 7日後解き直し |
| 出典 | 用語解説 | 定義確認 |

  • 例えば6月第2週火曜に早見表をノート先頭に作り
  • 水曜の関係法令10問演習後に誤答3数値を追記·木曜に用語解説で定義確認
  • と1週サイクルを回し

サイトの数値·期限早見表ハブと頻出数値リスト記事も参照してください。

3計算·割合問題の暗記ではなく確認

第二種の「計算」は複雑な公式より、単位·割合·引っかけ肢の読み取りが中心です。

| パターン | 例 | 対策 |
| --- | --- | --- |
| 割合 | 50%→半分 | 割合記事で確認 |
| 単位 | ppm·mg/m³ | 用語解説で定義 |
| 逆数 | 2倍·1/2 | 声出し計算 |
| 引っかけ | 数値±1 | 早見表と照合 |
| 時間 | 6か月測定 | 期限列に記載 |

  • たとえば労働衛生10問で「換気量300m³/hを600m³/hと誤読」したら
  • 単位列を早見表に追加し
  • 割合·パーセント問題記事の例題3問を24時間後に解き直し

数値問題の引っかけ記事で典型肢パターンも確認してください。

4科目別の数値暗記優先度

40点足切りを意識し、40点未満科目の数値から優先します。

| 科目 | 数値例（代表） | 演習目標 |
| --- | --- | --- |
| 関係法令 | 50人·3年·5年 | 10問4問正解 |
| 労働衛生 | WBGT·管理濃度 | 10問4問正解 |
| 労働生理 | 線量·心拍 | 10問4問正解 |
| 全体 | 180点·各40点 | 30問本番 |

  • 例えば9月第2週30問演習で関係法令42点·労働生理35点なら
  • 数値暗記の新規追加は労働生理のWBGT·休息基準から2語に絞り
  • 関係法令の数値は解き直し中心で維持し

労働衛生ハブ記事の8論点表とセットで優先順位を決めてください。

57日サイクルと直前2週間

数値暗記の出口は、7日後解き直しで正答率70%以上·30問本番で各40点以上です。

1. Day0：科目10問演習·数値誤答を記録
2. Day1：早見表追記·用語解説で定義
3. Day3：比較表で混同数値を3列整理
4. Day7：同論点10問解き直し
5. 直前2週：新規数値追加停止·★印解き直し

  • たとえば10月11日（日）13:30開始の2週前·9月27日（日）に★印数値15個を声出し10分
  • 9月28日（月）に関係法令10問で保存年数·人数基準2問正解を確認
  • と直前カレンダーを固定し

44問教材から拾った数値はこの段階で机から外してください。

6よくある質問

数式集を丸暗記すべきですか？
丸暗記より演習連動が効率的です。第二種30問では換気量等の計算より、18%·10ppm·50人·3年/5年のような数値·期限の一語差が頻出します。たとえば関係法令10問で数値3問中1問正解なら、誤答2数値だけ早見表に追記し7日後解き直し、の1週サイクルを3回回す方が40点足切りに効きます。頻出数値リスト記事で網羅確認もしてください。
第一種の数値リストは使えますか？
比較学習の参考程度に留め、第二種演習の誤答から作る早見表を正本にしてください。第一種44問·有害業務専門の濃度·手順は第二種出題範囲外です。例えば特定化学物質の深い管理濃度を第二種直前に暗記しても30問得点に直結しにくいです。第一種·第二種の違い記事で出題境界を再確認してください。
数値だけ覚えれば計算問題は不要ですか？
数値暗記だけでは不足です。割合·単位·主体がセットで問われます。たとえば「50%削減」を選ぶ問題では、数値50%の暗記に加え、元の量との関係を声出しで確認する習慣が必要です。割合·パーセント問題記事の例題5問と数値引っかけ記事を演習前に1周すると関係法令·労働衛生10問の得点が安定しやすくなります。
