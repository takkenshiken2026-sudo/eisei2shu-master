中国四国安全衛生技術センター【受験情報・アクセス確認·第二種衛生管理者】

第二種衛生管理者試験を中国四国安全衛生技術センターで受ける場合、会場·交通·試験日の正本は安全衛生技術試験協会の要項とセンター公式ページ（https://www.exam.or.jp/center_chushi/）です。本文には住所や最寄り駅は載せません。受験票の表記が本人の会場正本です。試験は30問·300点·総合180点·各科目40点·約3時間·13:30開始です。中国·四国在住者は申込時に定員を早めに確保し、遠方からの受験者は前日宿泊·移動時間を9月までに見積もるのが定番です。たとえば10月11日（日）13:30開始の回を中国四国センターで受けるなら、8月上旬の受付開始日にログインし、9月に公式アクセスで12:30到着目安を逆算してください。

この記事の要点

本記事を読むと、たとえば岡山在住で中国四国センターを選んだ場合、要項·定員·到着目安を公式リンクから確認し、受験票到着後に会場名を照合する手順が分かります。

  • center_chushiが正本
  • 先着定員
  • 遠方は宿泊検討
  • 12:30到着
  • 30問·13:30

1中四国センターの公式確認先

中国四国安全衛生技術センターの案内は、公式ページが正本です。住所·最寄り駅は本文に載せません。

| 確認先 | URL·資料 |
| --- | --- |
| センター公式 | exam.or.jp/center_chushi/ |
| 実施場所一覧 | exam.or.jp/h_aramashi/ |
| 試験要項 | exam.or.jp/ |
| 本人の会場 | 受験票 |
| 問い合わせ | 要項記載の連絡先 |

  • たとえば9月第2週に受験票が届いたら
  • 会場名を center_chushi/ の最新案内と照合し
  • 地図アプリで前日までにルート確認を済ませ

第二種は有害業務除外·30問·受験料8,800円です。

2先着定員と申込タイミング

中国四国センターも先着定員です。申込が遅れると満席·他地域への振替が必要になることがあります。

| フェーズ | 行動 | 注意 |
| --- | --- | --- |
| 6月 | 試験日をカレンダー登録 | 10月11日（日）等 |
| 8月上旬 | 受付開始日に申込 | 定員確保 |
| 8月中旬 | 受験料8,800円決済 | 締切前完了 |
| 9月 | 受験票到着待ち | 会場照合準備 |
| 10月直前 | 交通·宿泊確定 | 12:30到着 |

  • 例えば8月5日に中国四国センター満席なら
  • 近畿センターを第2希望にし
  • 10月11日（日）の前日宿泊·早朝出発が必要か当日中に判断してください

申込チェックリストで資格·写真も同時確認します。

313:30開始·30問·約3時間

中国四国センターでの第二種試験も、全国共通の試験形式です。要項が正本です。

| 項目 | 内容 |
| --- | --- |
| 問題数 | 30問（労働衛生·関係法令·労働生理） |
| 配点 | 1問10点·300点満点 |
| 合格 | 180点以上·各科目40点以上 |
| 時間 | おおむね3時間·13:30開始 |
| 出題外 | 有害業務（第一種向け） |

  • たとえば10月11日（日）13:30開始まで残り4週の9月13日なら
  • 新規テキストを止め
  • 30問演習の誤答20問を解き直す方が得点効率が高い

44問·100問用の第一種教材は第二種直前に使わないでください。

4当日の到着目安表

中国·四国エリアは移動に時間がかかるケースがあるため、到着目安を前日までに表にします。

| 到着目標 | 行動 | 備考 |
| --- | --- | --- |
| 12:00 | 会場到着 | 渋滞余裕込み |
| 12:15 | トイレ·水分補給 | 約3時間に備える |
| 12:30 | 受付完了 | 列待ち想定 |
| 12:45 | 筆記用具確認 | HB鉛筆2本等 |
| 13:30 | 試験開始 | 30問·5肢択一 |

  • 例えば広島から中国四国センターまで移動120分なら
  • 9:30出発·11:30到着·12:30受付完了を目標に
  • 前日に交通遅延時の予備ルートも控えておき

住所の断定はせず、公式ページと地図アプリで確認してください。

5持ち物と交通確認

試験当日は、要項·受験票の指示に従い持ち物を準備します。

1. 受験票·筆記用具（HB程度）·消しゴム
2. 身分証（要項·会場案内どおり）
3. スマホは指定場所に預ける想定
4. 前日20:00に持ち物リストへチェック
5. 公式アクセスで交通手段を最終確認

たとえば10月10日（土）20:00に①受験票②身分証③鉛筆2本④消しゴムをテーブルに並べ、10月11日（日）7:00に封筒へ入れます。当日持ち物記事·当日の流れ記事とあわせ、12:30到着までのタイムラインを1枚にまとめてください。

6よくある質問

中国四国センターのアクセスはどこで見ますか？
中国四国安全衛生技術センターの公式ページ（exam.or.jp/center_chushi/）と受験票が正本です。本文に住所は載せません。たとえば9月に受験票到着後、公式アクセスマップを開き、地図アプリで自宅（または宿泊先）から会場までの所要時間を前日までに確認してください。交通遅延時の予備ルートも前日に決めておくと安心です。
中国·四国以外から受験する場合の注意は？
遠方から中国四国センターを選ぶ場合、前日宿泊·早朝出発が必要になることがあります。例えば関東在住で10月11日（日）13:30開始·中国四国センター受験なら、10月10日（土）に宿泊し、当日10:00出発·12:30到着を前日カレンダーに書き込みます。申込時に定員を確保できたかも受験票で再確認してください。
試験当日の持ち物は何ですか？
受験票·筆記用具（HB程度の鉛筆またはシャープペン、消しゴム）が基本です。電子機器·辞書·参考書類は原則持込不可のため、安全衛生技術試験協会の受験要項で禁止物を確認してください。たとえば10月10日（土）夜に持ち物10項目チェックリストを実行し、10月11日（日）12:45到着後スマホを預けてから着席します。当日持ち物記事で封筒分区の詳細も確認してください。
