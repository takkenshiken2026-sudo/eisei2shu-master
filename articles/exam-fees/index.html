第二種衛生管理者試験の受験料

第二種衛生管理者試験の受験料は8,800円（非課税）です。たとえば10月11日（日）13:30開始の回を受けるなら、8月上旬の申込開始と同時に決済まで完了させる必要があり、残り18週の6月6日時点では支払方法と予算を先に確認しておくのが安全です。金額·支払手段の正本は安全衛生技術試験協会（https://www.exam.or.jp/）の要項です。

この記事の要点

本記事を読むと、1回8,800円の内訳と再受験·併願時の合計費用が分かり、たとえば春期·秋期の2回受験で17,600円になる点を申込前の予算表に書き込めます。

  • 8,800円（非課税）
  • 再受験も同額
  • 併願は試験ごと
  • 返金なし
  • 要項が正本

1受験料8,800円の内訳と正本

第二種衛生管理者試験の受験料は8,800円（非課税）です。年度ごとに要項で公表されるため、過去の記事や口コミではなく最新要項を開いて確認してください。

| 項目 | 内容 |
| --- | --- |
| 受験料 | 8,800円（非課税） |
| 対象 | 第二種衛生管理者試験1回分 |
| 正本 | 要項の「受験料」章 |
| 不合格時 | 返金なし |

  • たとえば2026年10月11日（日）13:30開始の回を受ける場合
  • 申込時に8
  • 800円を支払い
  • 受験資格不備で無効になっても原則返金されない点に注意してください

2支払方法と期限の確認

支払方法·支払期限は年度ごとに要項で定められます。申込画面の指示に従い、期限内に決済完了まで進めてください。

| 確認項目 | 方法 |
| --- | --- |
| 支払方法 | 要項記載（クレジット等） |
| 支払期限 | 申込締切日·時刻 |
| 控え | 完了画面·メールを保存 |
| 再確認 | 申込完了メールの到着 |

例えば締切日の17:00が受付終了時刻の年度で。

  • 最終日の午前中に8
  • 800円の決済まで完了させ
  • 午後は入力内容の再確認に充て

締切3日前までに支払いを済ませるのが無難です。

3再受験·併願時の費用

再受験·併願では試験ごとに8,800円が必要です。1回の申込で複数試験が割引になるわけではありません。

| パターン | 受験料（例） |
| --- | --- |
| 第二種のみ（1回） | 8,800円 |
| 春期·秋期の2回受験 | 17,600円 |
| 第二種＋別日程の他試験 | 8,800円×2以上 |
| 不合格後の再受験 | 毎回8,800円 |

たとえば2026年10月11日（日）の試験で不合格·翌年4月12日（日）回に再受験する場合、合計17,600円·別申込が必要です。併願ルール記事で日程衝突を確認してから予算を組んでください。

4支払い失敗時の対処

決済エラーは締切直前に多発します。事前の対処で取りこぼしを防ぎます。

| 症状 | 対処 |
| --- | --- |
| カードエラー | 別カード·支払方法を試す |
| タイムアウト | 混雑を避け早めに再試行 |
| 未完了画面 | 申込履歴で状態確認 |
| 不明 | 試験協会へ問い合わせ |

例えば締切7日前の8月4日に8,800円の決済を完了させ、最終日は入力内容の再確認だけに留めると安全です。決済完了前にブラウザを閉じると申込未完了になるため、完了メールが届くまで画面を確認してください。

5申込前の予算計画

受験料以外の費用も含め、申込前に1枚の表にまとめます。

1. 受験料8,800円×受験予定回数
2. 交通費·宿泊費（遠方会場の場合）
3. 公式テキスト·演習教材（任意）
4. 再受験を見込む場合は2回分17,600円を確保
5. 併願する他試験の受験料は別要項で確認

  • たとえば10月11日（日）13:30開始の第二種と
  • 翌年4月の再受験を見込むなら最低17
  • 600円を6月時点で確保し
  • 申込チェックリストと一緒に管理してください

6よくある質問

受験料はいくらですか？
第二種衛生管理者試験の受験料は8,800円（非課税）です。再受験のたびに同額が必要で、不合格でも返金はありません。例えば春期·秋期の2回受験する場合は合計17,600円になります。支払方法·期限は要項の「受験料」章が正本です。
不合格でも受験料は返金されますか？
原則返金されません。8,800円は1回の受験権に対する費用です。たとえば10月11日（日）の試験で150点（不合格）でも、翌年4月回に再受験する場合は改めて8,800円が必要です。受験資格不備で申込が無効になった場合も、返金がないことが多いため、申込前に資格を確認してください。
併願すると受験料は割引になりますか？
なりません。第二種衛生管理者試験は1回8,800円（非課税）で、併願しても試験ごとに同額が必要です。たとえば第二種10月回と作業環境測定士（別日程）の2試験を併願する場合、第二種分8,800円に加え、測定士の受験料（別要項）が別途かかります。申込前に予算表を作成し、支払期限も試験ごとに確認してください。
