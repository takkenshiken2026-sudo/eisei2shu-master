1管理体制の全体像

事業者 │ ├─ 総括安全衛生管理者（業種により100〜1,000人以上） │ ├─ 衛生管理者（50人以上・全業種） │ ├─ 安全管理者（製造業・建設業等の50人以上） │ └─ 産業医（50人以上・全業種） │ └─ 衛生委員会（50人以上・全業種） ├─ 議長（総括安全衛生管理者等） ├─ 衛生管理者 ├─ 産業医 └─ 労働者代表（半数）

2役職別 選任要件早見表

役職選任が必要な規模対象業種資格総括安全衛生管理者業種により100・300・1,000人以上業種限定業種により異なる衛生管理者常時50人以上全業種衛生管理者免許（1種・2種）安全管理者業種により50人以上製造業・建設業等安全管理者の資格産業医常時50人以上全業種医師衛生委員会常時50人以上全業種構成要件あり

3総括安全衛生管理者の業種別選任要件

業種選任が必要な規模林業・鉱業・建設業・運送業・清掃業常時100人以上製造業・電気業・ガス業・水道業・通信業・旅館業・ゴルフ場業等常時300人以上その他（情報通信・金融・小売業等）常時1,000人以上

4専任・専属の要件

条件内容衛生管理者の専任常時1,000人超の事業場：1人以上を専任産業医の専属常時1,000人以上または有害業務に500人以上従事：専属産業医が必要

5衛生委員会の構成

構成員規定議長総括安全衛生管理者等（事業者が指名）衛生管理者1名以上産業医1名以上労働者代表半数（労働組合等の推薦）開催頻度毎月1回以上議事録保存3年間

6試験で狙われる頻出ポイント

「全業種で100人以上なら総括安全衛生管理者が必要」→ 誤り（業種により300人・1,000人以上の場合がある）「産業医は医師でなくてもよい」→ 誤り（産業医は医師でなければならない）「衛生管理者の専任は500人超から必要」→ 誤り（1,000人超から）「衛生委員会は年1回開催すればよい」→ 誤り（毎月1回以上）
