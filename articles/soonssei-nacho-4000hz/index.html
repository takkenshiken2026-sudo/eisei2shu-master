第二種衛生管理者試験の騒音性難聴【4000Hz・TTS・PTS】試験に出る仕組みを整理

第二種衛生管理者試験で騒音性難聴が問われるとき、出題者は「4000Hz付近の聴力曲線」「TTS（一時的閾値シフト）」「PTS（永続的閾値シフト）」の3点を5肢択一の定義肢·因果肢·対策肢に分解して出します。合格は30問·300点·総合180点（60%）以上·各科目40点以上·3科目×各10問·有害業務除外·約3時間（10月11日（日）13:30開始）·受験料8,800円です。44問·100問は第一種の数値です。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、労働生理10問演習で「TTSは永久に回復しない」肢を誤選したら、本記事のTTS/PTS2列表→10問ドリル→soonnsei-nacho-4000hz記事の85dB表、の順で第7週を組みます。本記事は試験の出題メカニズムと演習ドリルに特化し、4000Hz·85dB·防音の網羅整理はsoonnsei-nacho-4000hz記事が正本です。正本は安全衛生技術試験協会（https://www.exam.or.jp/）要項です。

この記事の要点

本記事を読むと、たとえば労働生理10問で「4000Hz付近の聴力低下」肢を正解できても「TTSは休養後に回復しうる」肢を誤選した理由が出題パターン（定義肢）か演習不足かと切り分け、TTS/PTS10問ドリルに接続できます。

1騒音性難聴の出題メカニズム（3分解）

第二種30問では、騒音性難聴は次の3要素に分解されて出題されます。

| 要素 | 出題形式 | 主科目 |
| --- | --- | --- |
| 4000Hz付近低下 | 聴力曲線·特徴肢 | 労働生理10問 |
| TTS | 一時的·回復しうる | 労働生理10問 |
| PTS | 永続的·回復困難 | 労働生理10問 |
| 85dB·防音 | 暴露基準·対策肢 | 労働衛生10問 |
| 第二種 | 基礎理解 | 深い医学は出題外 |

  • たとえば労働生理10問で「低音域から低下」肢を誤選したら
  • 出題者は4000Hz付近·高音域
  • という定番パターンを使っているサイン

85dB·防音の網羅表はsoonnsei-nacho-4000hz記事を参照してください。

2TTS·PTSの設問パターン表

TTS/PTSは定義肢·因果肢の2パターンで繰り返し出ます。

| パターン | 正答の方向 | 典型誤答 |
| --- | --- | --- |
| TTS定義 | 一時的·回復しうる | PTSと混同 |
| PTS定義 | 永続的·回復困難 | TTSと混同 |
| 暴露後 | TTS→休養→回復 | 永久低下 |
| 長期暴露 | PTSへ移行 | TTSのまま |
| 管理 | 暴露低減·保護具 | 測定だけ |

  • 例えば「TTSは永久に回復しない」肢は誤りで
  • TTS一時的·PTS永続的
  • の2列をノートに書き
  • 労働生理10問演習で定義肢2問を24時間後に解き直すのが定番

34000Hz聴力曲線の出題肢と演習

4000Hz付近（3〜6kHz）の聴力低下は、聴力曲線の形として問われます。

| 肢の内容 | 正誤 | 確認方法 |
| --- | --- | --- |
| 4000Hz付近から低下 | 正 | 曲線スケッチ |
| 低音域から低下 | 誤 | 騒音性難聴と不一致 |
| 全周波数均等 | 誤 | 感音性の特徴と不一致 |
| 回復困難な進行 | 正（PTS文脈） | TTS/PTSとセット |
| 第二種 | 基礎 | 深い audiometry は出題外 |

  • たとえば労働生理10問で騒音2問中1問正解·4000Hz肢のみ誤答なら
  • 聴力曲線のスケッチ付き1行メモを週1回書き写し
  • 第8週演習前10分で確認し

労働生理まとめ記事の騒音性難聴行も参照してください。

4TTS/PTS10問ドリル（演習手順）

本記事の出口は、TTS/PTS定義肢10問のドリルです。

| 回 | 内容 | 目標 |
| --- | --- | --- |
| 1回目 | 解説付き5問 | 定義2列完成 |
| 24h後 | 同5問解説なし | 5問中4問正解 |
| 3日後 | 追加5問 | 10問中8問正解 |
| 1週後 | 労働生理10問混在 | 40点以上 |
| 2週連続 | 科目別10問 | 40点×2週 |

例えば第6週にTTS/PTS5問中2問正解·第7週24h後4問正解なら、ドリルは機能しています。

  • 第7週末の労働生理10問で38点なら
  • 4000Hz曲線スケッチを追加し
  • soonnsei-nacho-4000hz記事の85dB表と1枚に並べて復習してください

5soonnsei記事との役割分担

本記事（soonssei·typo slug）とsoonnsei-nacho-4000hz記事の分担です。

| 項目 | 本記事 | soonnsei記事 |
| --- | --- | --- |
| 目的 | 出題メカニズム·ドリル | 4000Hz·85dB網羅 |
| 粒度 | TTS/PTS設問パターン | 労働衛生·生理両角度 |
| 演習 | 10問ドリル手順 | 科目別演習接続 |
| 読者 | 定義肢で落ちる人 | 騒音論点全体復習 |

たとえば第7週にTTS/PTSドリル8/10正解·85dB肢2問中0問正解なら、本記事のドリルは完了としてsoonnsei-nacho-4000hz記事の85dB·防音表に週2時間を移してください。44問·100問用教材は第二種演習から外します。

6よくある質問

soonsseiとsoonnsei記事、どちらを先に読みますか？
TTS/PTS定義肢で2回連続誤答するなら本記事（出題メカニズム）が先、騒音論点全体を1枚に整理したいならsoonnsei-nacho-4000hz記事が先です。たとえば第6週労働生理10問で騒音2問中0問正解なら、本記事のTTS/PTS2列表→10問ドリル、第8週以降soonnsei記事の85dB表、の順が定番です。どちらも30問·各40点の合格基準は同じです。
TTS/PTSは何問くらい出題されますか？
第二種全体30問のうち、TTS/PTSを含む騒音関連は労働生理·労働衛生あわせておおむね2〜4問程度が目安です（年度で変動）。例えば過去演習20問中TTS/PTS定義肢2問·4000Hz曲線肢1問だったら、本記事の10問ドリルを週1回×2週で40点未満科目がある場合はその科目に時間を追加してください。
4000Hzは暗記だけで足りますか？
数値暗記に加え、聴力曲線の形（4000Hz付近から低下）をスケッチで確認できる必要があります。たとえば「4000Hz」だけ覚えて「低音域から低下」肢を選ぶ誤答は演習で繰り返し出ます。本記事の4000Hz表→10問ドリル→soonnsei-nacho-4000hz記事の労働衛生85dB表、の3段で定着させてください。
