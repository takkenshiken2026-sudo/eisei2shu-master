1用語解説で確認する内容

第二種衛生管理者試験では、免許申請は定義・要件・試験での引っかけがセットで問われます。長文の定義やFAQは <a href="/terms/menkyo-shinsei.html">用語解説「免許申請」</a> にまとめています。試験ガイド本文に同じ定義を繰り返さないのが、このサイトの配置ルールです。

2試験ガイドとの使い分け

用語解説は「免許申請**とは何か**」を答える場所です。一方、勉強の進め方・申込・直前対策・再受験は <a href="/articles/index.html">試験ガイド</a> 側で扱います。定義を読んだら、弱点分野のガイドか過去問演習へ進んでください。

3おすすめの学習順

① <a href="/terms/menkyo-shinsei.html">用語解説</a> で定義と誤答パターンを確認。② 関連用語リンクから混同語を1周。③ 過去問で該当分野を解き、間違えた選択肢を用語解説で照合。④ 必要なら学習計画・分野別対策ガイドで復習サイクルを組みます。

4よくある質問

免許申請の意味はどこで詳しく読めますか？
<a href="/terms/menkyo-shinsei.html">用語解説「免許申請」</a> が正本です。定義・試験論点・FAQ・例題を1ページにまとめています。試験ガイド側には同じ本文を載せないため、リンクから知識ハブへ移動してください。
用語を読んだあと何をすればよいですか？
関連用語を2件以上確認したうえで、過去問演習に進むのが基本です。学習全体の設計は試験ガイドの「学習計画」「分野別対策」を参照してください。直前・再受験の悩みは注意点・更新ジャンルの記事が該当します。
試験ガイドと用語解説の違いは？
用語解説は知識（What）、試験ガイドは進め方（How/When）です。免許申請 のような用語の意味は用語解説、勉強スケジュールや申込手続きは試験ガイドに置きます。
