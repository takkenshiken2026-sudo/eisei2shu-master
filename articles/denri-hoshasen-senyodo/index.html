電離放射線·線エネルギー【5年100mSv·3ヶ月1.3mSv】第二種整理

電離放射線の線量限度·管理区域·線エネルギー（LET）は、第二種衛生管理者試験の関係法令·労働生理双方にまたがる論点です。第二種30問では基本の数値·区域管理·ALARA·確定的/確率的影響の比較が中心で、有害業務（放射性物質取扱い等）の深掘りは第一種寄りのため出題外です。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、6月第2週に5年100mSv·3ヶ月1.3mSv·管理区域の数値表を転記し、8月第1週に関係法令10問演習で放射線2問正解を目標にする流れが定番です。正本は放射線障害防止関係法令·公式テキスト·安全衛生技術試験協会（https://www.exam.or.jp/）要項です。

この記事の要点

本記事を読むと、たとえば関係法令10問で「5年100mSv·3ヶ月1.3mSv·年50mSv」の組合せ肢を整理でき、有害業務除外の範囲でどこまで学ぶかの線引きが表で分かります。

  • 5年100mSv
  • 3ヶ月1.3mSv
  • 管理区域
  • 線エネルギー
  • 有害業務除外

1線量限度の数値表（第二種出題範囲）

職業被ばくの線量限度は数値セットで問われます。詳細の正本は用語解説·数値早見表です。

| 指標 | 代表値 | 試験での注意 |
| --- | --- | --- |
| 5年 | 100mSv（実効線量） | 期間の積算 |
| 1年 | 50mSv（上限） | 単年との組合せ |
| 3ヶ月 | 1.3mSv（等価線量·眼の水晶体等） | 部位別 |
| ALARA | 限度内でも低減 | 努力義務 |
| 管理区域 | 立入·標識 | 区域と限度は別 |

たとえば8月第1週火曜19:00〜19:20に上表をノートへ転記し、水曜の関係法令10問演習で「3ヶ月1.3mSv·5年100mSv·年50mSv」の組合せ肢2問の正答理由を書き留めます。第一種100問教材の放射性物質深掘りは第二種直前に読まないでください。

2管理区域と線エネルギーの関係

管理区域は線量·汚染のおそれに応じた立入管理、線エネルギー（LET）は放射線の種類と生物学的影響の違いを整理する概念です。

| 概念 | 内容 | 混同注意 |
| --- | --- | --- |
| 管理区域 | 立入制限·標識·測定 | 作業場名称ではない |
| 周辺監視区域 | 管理区域外の監視 | 区域の内外 |
| 線エネルギー | α線高·γ線低（代表） | 透過力とセット |
| 外部·内部被ばく | 経路の違い | 区域·換気 |

  • 例えば関係法令10問で「管理区域に設定すれば個人防護具不要」肢が出たら
  • 区域管理とPPEは併用が原則
  • と用語解説で確認し

確定的·確率的影響記事とセットで放射線3点（限度·区域·影響）を復習してください。

3有害業務除外と比較学習の線引き

第二種試験は30問·有害業務除外が要項どおりです。放射線の基本（限度·区域·ALARA·影響分類）は出ますが、放射性物質取扱いの細部·第一種レベルの有害業務管理は比較論点に留めます。

| 学習する（第二種） | 比較のみ（第一種寄り） |
| --- | --- |
| 5年100mSv·3ヶ月1.3mSv | 詳細な取扱規程 |
| 管理区域·標識 | 放射性廃棄物 |
| ALARA·三原則 | 内部被ばく深掘り |
| 確定的/確率的 | 有害業務認定細目 |

  • たとえば6月6日時点で第一種100問教材を持っている場合
  • 放射線章のうち線量限度·管理区域の10ページだけ第二種ノートに転記し
  • 残りは試験後まで封印するのが定番

44問·100問の合格基準（240点等）で学習時間を配分しないでください。

4第一種との比較論点（第二種受験者向け）

第一種情報と混同しやすい論点を、第二種30問で必要な範囲に切り取ります。

| 論点 | 第二種で押さえる | 第一種深掘り |
| --- | --- | --- |
| 試験問題数 | 30問 | 44問 |
| 放射線 | 限度·区域·ALARA | 有害業務細目 |
| 合格点 | 180点·各40点 | 240点等 |
| 学習時間 | 60時間前後目安 | 別計画 |

  • 例えば9月第2週に関係法令10問で放射線2問·労働生理10問で線量指標1問
  • 合計3問出たら
  • 第二種範囲内として3問すべて正解を目標にし
  • 第一種過去問の放射性物質10問追加は第二種直前には控え

第一種·第二種の違い記事で出題範囲表を再確認してください。

5関係法令10問·労働生理10問への接続

放射線論点は関係法令と労働生理に分散します。科目別10問で40点足切りを確認しながら学習します。

1. 数値表（5年100mSv·3ヶ月1.3mSv）を転記
2. 用語解説で管理区域·ALARAを確認
3. 関係法令10問演習で放射線2問
4. 労働生理10問で線量指標1問
5. 7日後に★印解き直し

  • たとえば10月4日（日）の30問本番形式前に
  • 9月20日（土）に関係法令10問·労働生理10問を科目別に解き
  • 放射線論点3問の正答率を記録し

180点·各40点の足切りラインを下回った科目は数値表の声出し復習を週2回に増やしてください。

6よくある質問

5年100mSvと3ヶ月1.3mSvは両方覚えますか？
はい、セットで問われることがあります。5年100mSvは実効線量の期間限度、3ヶ月1.3mSvは部位別（眼の水晶体等）の代表値として整理します。たとえば8月第1週に数値表を転記し、関係法令10問演習で組合せ肢2問の正答理由をノートに書くのが定番です。数値早見表·頻出数値リスト記事と突合し、試験直前は★印のみ声出し10分で確認してください。
線エネルギーは労働生理でも出ますか？
出ることがあります。α線は線エネルギーが高く透過力が低い、γ線は相対的に低い線エネルギーで透過力が高い、の典型对比が問われます。例えば労働生理10問で外部·内部被ばくとセットの肢が出たら、用語解説の線量指標（吸収·等価·実効）と3列比較表を書き、7日後に同論点10問を解き直します。関係法令の管理区域論点と混同しないよう科目別に記録してください。
有害業務の放射線は第二種で学びますか？
深掘りは第二種出題外です。要項どおり有害業務は除外され、線量限度·管理区域·ALARA·影響分類など基本論点が第二種30問の範囲です。たとえば第一種100問教材の放射性物質章全20問を第二種直前に解いても、30問·各40点足切りの演習記録は改善しにくいです。比較学習として第一種情報を「どこまで不要か」を線引きし、基本数値表に集中してください。
