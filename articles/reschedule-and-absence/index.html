第二種衛生管理者試験の欠席・再受験の手続き

第二種衛生管理者試験を欠席した場合、その回の受験権は失効し、受験料8,800円（非課税）の返金は原則ありません。たとえば10月11日（日）13:30開始の回を体調不良で欠席した場合、翌年4月回へ再申込するには2月頃から先着受付·8,800円決済が必要です。本記事では欠席時の扱いと再受験申込の流れを表にまとめます。正本は常に安全衛生技術試験協会（https://www.exam.or.jp/）の最新要項です。

この記事の要点

本記事を読むと、たとえば10月11日（日）13:30開始の回を体調不良で欠席した場合に何が起き、翌年4月回へ再申込する際の2か月前スケジュールが分かります。

  • 欠席＝受験権失効
  • 返金なし
  • 再申込8,800円
  • 2か月前先着
  • 要項が正本

1欠席した場合の基本扱い

  • 試験当日に会場へ出ず欠席した場合
  • その回の受験は「未受験」扱いとなり
  • 合否判定の対象になりません

申込時に支払った8,800円は原則返金されません。

| 状況 | 扱い |
| --- | --- |
| 当日欠席 | 未受験·返金なし |
| 不合格 | 合否判定あり·再受験可 |
| 次回受験 | 改めて申込·8,800円 |
| 確認先 | 要項·試験協会 |

  • たとえば10月11日（日）13:30開始の回を体調不良で欠席した場合
  • 合格発表日に合否を確認する必要はなく
  • 次回試験日（春期4月頃または秋期10月頃）の申込準備に切り替え

試験日変更が可能かは年度ごとの要項で確認してください。

2試験日変更·キャンセルの可否

試験日の変更·申込キャンセルができるかは、年度·申込時期·要項の定めにより異なります。欠席前に変更手続が間に合う場合もあるため、やむを得ず受験できないときは早めに要項を確認します。

| 手続 | 確認ポイント |
| --- | --- |
| 受験地変更 | 定員·期限内 |
| 試験日変更 | 要項に記載があるか |
| キャンセル | 返金の有無·期限 |
| 問い合わせ | 試験協会窓口 |

  • 例えば10月11日（日）13:30開始の2週間前に怪我で受験不可と判明した場合
  • 要項の変更·取消の章を開き
  • 期限内手続があるか試験協会へ問い合わせ

間に合わない場合は欠席扱い·再申込（8,800円）を前提に計画します。

3再受験申込の流れ

欠席·不合格のいずれも、次回受験は改めてオンライン申込が必要です。手順は初回申込と同様で、受験資格·写真·8,800円·受験地を確認します。

| ステップ | 内容 |
| --- | --- |
| 1 | 次回試験日をexam.or.jpで確認 |
| 2 | 2か月前から先着申込 |
| 3 | 8,800円決済·受験地確保 |
| 4 | 受験票到着·13:30開始確認 |
| 5 | 30問·3科目の学習再開 |

  • たとえば10月11日欠席後
  • 翌年4月の春期回を受けるなら
  • 2か月前（2月頃）の受付開始日にログインし
  • 1週間以内に決済完了するのが定番

申込みの流れ記事で6ステップの詳細を確認してください。

4欠席·再受験の費用

再受験·欠席後の再申込は、いずれも8,800円（非課税）が必要です。併願割引や不合格割引はありません。

| パターン | 費用（例） |
| --- | --- |
| 初回受験 | 8,800円 |
| 欠席後の再申込 | 8,800円 |
| 不合格後の再受験 | 8,800円 |
| 春期·秋期2回 | 17,600円 |

  • 例えば10月11日（日）13:30開始を欠席し
  • 翌年4月回·同年10月回の2回再挑戦する場合
  • 合計17
  • 600円＋初回8
  • 800円＝26
  • 400円の予算が必要

6月時点で予算表に書き、受験料記事で支払方法を確認してください。

5欠席後の学習再開

欠席後は、次回試験日までの学習計画を立て直します。30問·300点·180点·各科目40点の基準は変わりません。

1. 次回試験日と2か月前申込日をカレンダー登録
2. 30問演習で3科目得点を記録し40点未満科目を特定
3. 44問·100問は第一種（第二種は30問）と混同しない
4. 有害業務は出題範囲外（比較学習のみ）
5. 再受験日程と学習量のバランスを別記事で調整

  • たとえば10月欠席·翌年4月再受験なら
  • 11〜12月に誤答解き直し·1〜3月に30問本番形式演習2回
  • という流れが定番

日程調整の詳細は再受験の日程調整記事を参照してください。

6よくある質問

試験当日に欠席するとどうなりますか？
その回は未受験扱いとなり、合否判定はありません。申込時の8,800円は原則返金されません。たとえば10月11日（日）13:30開始の回を欠席した場合、次回は改めて申込·決済が必要で、希望試験日の2か月前から先着受付が始まります。試験日変更が可能かは要項で確認し、不明点は安全衛生技術試験協会へ問い合わせてください。
不合格と欠席の再受験手続きは同じですか？
次回申込の流れ（資格確認·写真·8,800円·受験地選択）は同じです。不合格の場合は合否確認後に弱点科目を特定し、たとえば労働生理38点なら40点底上げを優先します。欠席の場合は演習再開のタイミングから計画を立て直します。いずれも2か月前先着申込·返金なしを前提に、試験日程記事で次回回を確認してください。
欠席後、すぐ再申込できますか？
次回試験日の受付開始（おおむね2か月前）まで待つ必要があります。例えば10月11日欠席後、翌年4月の春期回を受けるなら、2月頃の受付開始日に申込みます。待機期間は30問演習·誤答解き直しに充て、180点·3科目40点を演習で記録できる状態を目指してください。申込チェックリスト記事で書類·写真を事前準備しておくとスムーズです。
