第二種衛生管理者試験の制度改定の見方

第二種衛生管理者試験の制度改定は、SNSや旧教材の数字より安全衛生技術試験協会の最新要項を確認してください（要項で再確認）。本記事は要項改定の読み方·5分チェック表·44問除外·演習記録更新·再受験者の見逃しパターンに特化しています。詳細は各関連記事を参照してください。例えば2026年10月11日の前に本記事の表を転記し、1週間後に同論点を解き直してください。

1改定情報の正本（安全衛生技術試験協会 試験実施場所（公式））

制度改定の第一確認先は安全衛生技術試験協会です（要項で再確認）。試験日程·受験資格·出題範囲·合格基準·受験料は年度ごとに要項で公表されます。

確認項目第二種の代表値
問題数30問
満点300点
合格180点·各40点以上
時間約3時間·13:30開始
受験料8,800円（非課税）

たとえば6月11日に要項を開き、「100問5科目」の記述がないことを確認できれば、第二種用の演習に絞れます。第一種44問の情報は比較学習のみに留めてください。

2よく更新される章と学習への影響

改定の影響が大きい章と、第二種学習への反映優先度を整理します（要項で再確認）。

要項の章改定時の対応
試験科目·出題範囲テキスト目次と照合
合格基準演習記録のライン修正
試験日程カレンダー再登録
受験資格申込書類の再確認
受験料予算表の更新

例えば8月に出題範囲の脚注が改定された場合、9月1日〜9月7日は新規演習より変更章のテキスト読み直しに6時間を割くのが定番です。有害業務除外は第二種の固定事項として、改定があっても要項の「試験科目」章で再確認してください。

344問·100問誤情報の除外手順

制度改定記事を読む前に、第一種の旧数値が混ざった情報を除外します（要項で再確認）。

誤情報例第二種の正
44問·400点30問·300点
100問5科目30問3科目
240点合格180点·各40点
300時間勉強60時間前後目安

たとえば7月11日に購入した問題集表紙が「100問完全対策」なら、10月11日（日）受験の主教材から外し、30問演習と公式テキストに切り替えます。第一種·第二種の違い記事で比較表を印刷し、机の横に貼っておくと改定記事の誤読が減ります。

4改定後の演習記録の更新

要項改定後は、演習ノートの見出しを第二種基準に揃え直します（要項で再確認）。

記録項目書き方（例）
日付2026-09-14（日）13:30模擬
総合186/300
労働衛生52/100
関係法令48/100
労働生理86/100

例えば改定前ノートが「60％ライン180点」だけなら、各科目40点欄を3列追加し、9月14日からの30問演習で「労働生理38点→不合格圏内」と明示記録するよう更新してください。合格基準記事で足切りの制度説明もあわせて確認します。

5再受験者が改定を見逃すパターン

再受験者は前回の要項記憶で改定に気づかないことがあります（要項で再確認）。

1. 前回合格基準だけ暗記·各40点を見落とす
2. 44問教材をそのまま使い30問配分が合わない
3. 有害業務章に時間を割き3科目が偏る
4. 受験料改定を知らず8,800円以外を準備
5. 試験日変更をカレンダー未更新

具体例として2025年10月不合格·2026年10月11日再受験なら、6月11日に要項PDFを前回ファイルと差し替え比較し、変更章に付箋を貼ってから3か月計画を立てるのが定番です。再受験戦略記事で学習期間の調整も確認してください。

6よくある質問

制度改定はどこで確認しますか？
安全衛生技術試験協会（安全衛生技術試験協会 試験案内（公式））の最新要項と試験案内を確認してください（要項で再確認）。一例として6月11日時点では、30問·3科目×各10問·180点·各40点·8,800円·10月11日（日）13:30開始·約3時間、を要項表から転記して教材選びに使います。ブログの「100問対策」記事は第二種にはそのまま使えません。公式情報確認先記事も参照してください。
改定後すぐ教材を買い替えますか？
公式テキストの改訂版が出た章だけ優先します（要項·版で再確認）。例えば関係法令章だけ改訂なら、8月1日〜8月7日にその章と関係法令10問演習20問をセットで解き、他2科目は旧版のまま9月まで使う分割も現実的です。44問専用問題集は第二種30問演習には不向きです。教材版更新記事も参照できます。
有害業務の範囲改定は第二種に影響しますか？
第二種では有害業務に関する出題はもともと範囲外です（要項で再確認）。例として特定化学物質の専門管理が第一種要項で改定されても、第二種学習の中心（3科目×各10問·有害除く）は変わりません。出題範囲概要記事で3科目の境界を再確認してください。
