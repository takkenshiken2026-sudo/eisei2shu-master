第二種衛生管理者試験の制度改定の見方【要項·出題範囲の更新】

第二種衛生管理者試験の制度改定は、SNSや旧教材の数字より安全衛生技術試験協会の最新要項が正本です。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、6月第1週に要項PDFの「試験科目」「合格基準」「受験料」を30問·300点·180点·各40点·8,800円と照合し、44問·100問の記述が混ざっていないか教材ラベルを洗い出すのが定番です。本記事では改定情報の読み方と、学習計画への反映順序を表にまとめます。

この記事の要点

本記事を読むと、たとえば要項改定で「各科目40点」が変わっていないかを5分チェックし、古い100問教材を第二種学習から外す判断ができます。

  • 要項が正本
  • 30問·3科目
  • 44問は第一種
  • 改定は章単位確認
  • 月1回再確認

1改定情報の正本（exam.or.jp）

制度改定の第一確認先は安全衛生技術試験協会です。試験日程·受験資格·出題範囲·合格基準·受験料は年度ごとに要項で公表されます。

| 確認項目 | 第二種の代表値 |
| --- | --- |
| 問題数 | 30問 |
| 満点 | 300点 |
| 合格 | 180点·各40点以上 |
| 時間 | 約3時間·13:30開始 |
| 受験料 | 8,800円（非課税） |

  • たとえば6月6日に要項を開き
  • 「100問5科目」の記述がないことを確認できれば
  • 第二種用の演習サイトだけに絞れ

第一種44問の情報は比較学習のみに留めてください。

2よく更新される章と学習への影響

改定の影響が大きい章と、第二種学習への反映優先度を整理します。

| 要項の章 | 改定時の対応 |
| --- | --- |
| 試験科目·出題範囲 | テキスト目次と照合 |
| 合格基準 | 演習記録のライン修正 |
| 試験日程 | カレンダー再登録 |
| 受験資格 | 申込書類の再確認 |
| 受験料 | 予算表の更新 |

例えば8月に出題範囲の脚注が改定された場合、9月第1週は新規演習より変更章のテキスト読み直しに6時間を割くのが定番です。有害業務除外は第二種の固定事項として、改定があっても要項の「試験科目」章で再確認してください。

344問·100問誤情報の除外手順

制度改定記事を読む前に、第一種の旧数値が混ざった情報を除外します。

| 誤情報例 | 第二種の正 |
| --- | --- |
| 44問·400点 | 30問·300点 |
| 100問5科目 | 30問3科目 |
| 240点合格 | 180点·各40点 |
| 300時間勉強 | 60時間前後目安 |

  • たとえば7月に購入した問題集表紙が「100問完全対策」なら
  • 第二種10月11日（日）受験の主教材から外し
  • 30問演習サイトと公式テキストに切り替え

第一種·第二種の違い記事で比較表を印刷し、机の横に貼っておくと改定記事の誤読が減ります。

4改定後の演習記録の更新

要項改定後は、演習ノートの見出しを第二種基準に揃え直します。

| 記録項目 | 書き方（例） |
| --- | --- |
| 日付 | 2026-09-14（日）13:30模擬 |
| 総合 | 186/300 |
| 労働衛生 | 52/100 |
| 関係法令 | 48/100 |
| 労働生理 | 86/100 |

  • 例えば改定前ノートが「60％ライン180点」だけなら
  • 各科目40点欄を3列追加し
  • 9月第2週の30問演習から「労働生理38点→不合格圏内」と明示記録するよう更新してください

合格基準記事で足切りの制度説明もあわせて確認します。

5再受験者が改定を見逃すパターン

再受験者は前回の要項記憶で改定に気づかないことがあります。

1. 前回合格基準だけ暗記·各40点を見落とす
2. 44問教材をそのまま使い30問配分が合わない
3. 有害業務章に時間を割き3科目が偏る
4. 受験料改定を知らず8,800円以外を準備
5. 試験日変更をカレンダー未更新

  • たとえば2025年10月不合格·2026年10月11日再受験なら
  • 6月第1週に要項PDFを前回ファイルと差し替え比較し
  • 変更章に付箋を貼ってから3か月計画を立てるのが定番

再受験戦略記事で学習期間の調整も確認してください。

6よくある質問

制度改定はどこで確認しますか？
安全衛生技術試験協会（https://www.exam.or.jp/）の最新要項と試験案内が正本です。たとえば6月6日時点では、30問·3科目×各10問·180点·各40点·8,800円·10月11日（日）13:30開始·約3時間、を要項表から転記して教材選びに使います。ブログの「100問対策」記事は第二種にはそのまま使えません。
改定後すぐ教材を買い替えますか？
公式テキストの改訂版が出た章だけ優先します。例えば関係法令章だけ改訂なら、8月第1週にその章と関係法令10問演習20問をセットで解き、他2科目は旧版のまま9月まで使う分割も現実的です。44問専用問題集は第二種30問演習には不向きです。
有害業務の範囲改定は第二種に影響しますか？
第二種では有害業務に関する出題はもともと範囲外です。たとえば特定化学物質の専門管理が第一種要項で改定されても、第二種学習の中心（3科目×各10問·有害除く）は変わりません。出題範囲概要記事で3科目の境界を再確認してください。
