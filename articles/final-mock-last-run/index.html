1直前模試の目的と10/4（日）の実施日

直前模試の目的は、本番と同じ条件で時間配分·集中力·得点を測ることです。第二種は30問·300点·おおむね3時間·13:30開始です（要項で再確認）。

項目内容
実施日10/4（日）など試験4〜7日前
問題数30問（3科目×各10問）
時間約180分·13:30開始想定
形式5肢択一マークシート

たとえば10/11（日）本番なら、10/4（日）13:30から210問演習の30問セットを開始し、16:30頃に終了する流れを体験します。有害業務は出題範囲外です。44問·100問は第一種の数値です。模試の活用法記事で年間の模試回数も確認できます。

230問本番形式の実施手順

模試は本番に近い環境で実施します。スマホは別室·電源OFFが理想です。

時刻内容
12:45着席·HB鉛筆2本準備
13:3030問開始·タイマー180分
16:20見直し10分
16:30採点·科目別得点記録

例えば10/4（日）13:30開始なら、労働衛生60分·関係法令60分·労働生理50分·見直し10分の配分で進めます。試験当日の時間配分記事の表と照合してください。分からない問題は仮マークして後回しにし、最終10分で未記入欄をチェックします。657問一問一答で科目別の弱点確認も並行できます。

3模試結果の読み方（180点·40点足切り）

採点後は、総合点と科目別40点足切りの両方を確認します。

得点判定の目安
総合180点以上合格ライン到達
各科目40点以上足切りクリア
1科目38点不合格リスク大
総合210点·1科目35点足切り不合格

たとえば10/4（日）模試で関係法令52点·労働衛生48点·労働生理38点（合計138点）なら、労働生理が40点足切りを下回っており、10/5（月）〜10/10（土）は労働生理の誤答解き直しに時間を集中します。合格基準と足切り記事で240点·60%等の第一種数値と混同しないよう注意してください。

4模試後24時間の復習

模試翌日までに、誤答15問を解き直すのが定番です。新規インプットより、模試で落とした論点の定着を優先します。

時間内容
模試後2時間誤答理由を5分以内に分類
10/5（月）60分誤答15問を解き直し
10/6（火）40点未満科目を1章確認
10/7（水）以降軽い確認·範囲絞り込み

例えば10/5（月）の60分で、10/4（日）模試の労働生理誤答7問を用語解説と照合し、10/8（木）にもう一度解き直します。範囲絞り込み記事の優先順位表と合わせて計画を更新してください。10/10（土）以降は前日チェックリスト記事に切り替えます。

5直前模試のよくある誤り

直前模試でよくある誤りは次のとおりです。

1. 10/10（土）前日に30問模試を実施し、睡眠·集中力が落ちる
2. 15問だけ解いて「本番形式」と称する
3. 44問·100問の第一種演習で時間感覚を練習する
4. 科目別得点を記録せず、総合点だけ見る
5. 模試後に誤答を解き直さず新教材を読み始める

たとえば10/11（日）本番なら、10/4（日）13:30開始·30問·180分で1回実施し、10/5（月）〜10/9（金）は誤答解き直し·範囲絞り込み、10/10（土）は前日チェックリストに切り替える流れが定番です。2回目の模試を10/7（水）に追加する場合も、13:30開始·30問形式を維持してください。

6よくある質問

直前模試は何回受けるべきですか？
本番形式（30問·約3時間）で1〜2回が目安です。たとえば10/4（日）と10/7（水）に13:30開始で各1回、2回目は1回目の誤答が減っているかを確認します。3回以上は睡眠·体調を削るため、10/8（木）以降は増やしすぎないでください。15問だけの演習では時間配分（6分/問）を体験できません。時間計測演習記事で年間の積み上げも確認できます。
模試で180点未満でも大丈夫ですか？
180点未満なら、10/5（月）〜10/10（土）で底上げ可能かを科目別に判断します。例えば総合165点·労働生理42点·関係法令40点·労働衛生38点なら、労働衛生の誤答10問を優先解き直し、40点ラインを先に確保する計画に切り替えます。1科目でも38点以下が続く場合は、足切りリスクが高いため範囲絞り込み記事の優先順位表を参照してください。
自宅と会場、どちらで受けるべきですか？
可能なら本番と同じ13:30開始·着席姿勢で受けるのが理想です。たとえば10/4（日）13:30に自宅の机で30問演習し、スマホは別室·通知OFFにします。会場までの移動は模試に含めず、試験当日の流れ記事で12:45到着の体験は別日に確認してください。マークシートに近い答案用紙があるとより本番に近づきます。
