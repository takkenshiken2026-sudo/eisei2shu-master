第二種衛生管理者試験のテキストの選び方

第二種衛生管理者試験のテキスト選びで最初に決めるのは「安全衛生技術試験協会の公式テキスト1冊を正本にする」ことです。市販解説本·通信講座テキストは補助で、表紙·目次が30問·3科目×各10問·有害業務除外に対応しているかを確認します。合格は総合180点（60%）以上·各科目40点以上·受験料8,800円·約3時間（10月11日（日）13:30開始）です。44問·100問は第一種の数値です。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、Day1に公式テキスト1冊を購入·目次で3科目章対応表を1枚に書き·100問問題集は購入リストから外す、の3手順で始めます。本記事はテキスト選びに特化し、問題集は問題集選び·テキストvs過去問は優先順位記事へ案内します。正本は安全衛生技術試験協会（https://www.exam.or.jp/）要項です。

この記事の要点

本記事を読むと、たとえば書店で「第二種30問テキスト」と「衛生管理者100問完全版」を見分け、前者だけを正本として選べます。

  • 公式テキスト正本
  • 30問3科目
  • 100問除外
  • 版照合
  • 1冊から開始

1公式テキストを正本にする理由

第二種のテキスト選びは、試験協会の公式テキストを起点にします。

| 教材 | 位置づけ | 第二種で |
| --- | --- | --- |
| 公式テキスト | 正本 | 必須1冊 |
| 市販解説本 | 補助 | 0〜1冊 |
| 通信講座テキスト | 補助 | 講座受講時 |
| 100問問題集 | 第一種 | 除外 |
| 一問一答 | 数値補助 | 本サイト657問 |

  • たとえば6月6日Day1に公式テキストを購入し
  • 目次を3科目（労働衛生·関係法令·労働生理）に色分けすると
  • 60時間独学計画の骨格が1枚で見え

公式情報確認先記事で要項PDFの開き方も確認してください。

230問3科目版の見分け方（目次チェック）

市販テキスト·解説本は、目次で第二種向きか確認します。

| 目次の特徴 | 第二種OK | 第一種NG |
| --- | --- | --- |
| 科目数 | 3科目 | 4〜5科目 |
| 問題数表記 | 30問 | 100問·44問 |
| 有害業務 | 除外·薄い | 厚い章 |
| 時間 | 約3時間 | 5時間 |
| 得点 | 180点·各40点 | 60%のみ |

例えば「衛生管理者試験テキスト100問完全攻略」は第一種向けのことが多く、第二種独学60時間計画に載せると出題外章に時間を使います。1shu-2shu-chigai記事で科目数の違いも確認してください。

3版·発行年と要項PDFの照合

テキスト購入後、奥付の版·発行年を要項PDFと照合します。

| 確認箇所 | 照合先 | ズレた場合 |
| --- | --- | --- |
| 奥付·版数 | 要項改定年 | 新版購入 |
| 出題範囲表 | 要項PDF | 差分章優先 |
| 数値·条文 | テキスト本文 | 演習で確認 |
| 中古テキスト | 最新要項 | 差分のみ読む |

  • たとえば2024年版テキスト·2026年10月回受験なら
  • 要項改定がなければ買い替え不要·改定があれば差分章だけ先に15分読む
  • が定番

教材版更新サイクル記事で買い替え判断の詳細も参照してください。

4初学者の最小構成（テキスト1冊）

初学者はテキスト1冊から始め、演習は本サイト210問で補います。

| 段階 | 教材 | 週 |
| --- | --- | --- |
| Day1 | 公式テキスト購入 | 第1週 |
| 第1〜4週 | 3科目骨格1周 | テキスト中心 |
| 第5週〜 | 科目別10問演習 | テキスト＋演習 |
| 第8週〜 | 30問本番形式 | 演習中心 |
| 直前4週 | 新規購入停止 | 解き直し |

例えば第4週までにテキスト3科目を薄く1周·第5週科目別10問で関係法令42点·労働生理38点なら、テキスト追加より労働生理2章読み直し＋演習10問の方が40点足切りに効きます。テキストと過去問優先順位記事で読む順も確認してください。

5テキスト選びと問題集選びの分担

テキスト（インプット）と問題集（演習）は別記事で選びます。

| 項目 | textbook-selection（本記事） | 接続先 |
| --- | --- | --- |
| 対象 | 公式·市販テキスト | 問題集選び記事 |
| 正本 | 試験協会テキスト | 過去問30問/年 |
| 冊数 | 1冊から | ＋演習1セット |
| 100問 | 除外 | 問題集側でも除外 |

たとえばDay1テキスト1冊·第5週30問過去問1セット、の2段購入が定番です。3冊同時購入（テキスト＋100問＋一問一答3冊）は読み切れず演習が遅れます。独学合格ガイド記事の最小構成4点も参照してください。

6よくある質問

公式テキスト以外は本当に不要ですか？
公式テキストは必須で、それ以外は弱点補強用です。たとえば関係法令10問で4問正解（40点）以上なら市販法令テキストは不要なことが多いです。逆に労働衛生10問で3問正解（30点）が2回続くなら、労働衛生まとめ記事と演習解き直しを先に回してください。問題集選びは別記事で30問版を確認してください。
通信講座のテキストだけで足りますか？
講座付属テキストは補助として使い、公式テキスト1冊は別途正本として確保してください。例えば講座テキストで骨格を学び、公式テキスト該当章で条文確認·日曜13:30から30問演習、と役割分担すると得点記録が正確になります。講座が100問形式中心なら第二種30問演習を本サイト210問で補完してください。
テキストは何章まで読めば演習に移れますか？
目次1周·3科目の骨格章（各分野の入門章）を読んだら科目別10問演習に移るのが定番です。たとえば第4週末時点で3科目骨格1周·第5週科目別10問で40点未満科目を特定、の流れが現実的です。全章完読後に演習開始だと60時間計画では演習回数が不足します。テキストと過去問優先順位記事も参照してください。
