第二種衛生管理者試験の関係法令の頻出論点

第二種衛生管理者試験の関係法令は30問中10問·各100点満点の1科目で、各科目40点以上の足切りに直結します。労働安全衛生法と規則を横断し、選任·健康診断·作業環境·化学物質·長時間労働など12前後の論点が繰り返し出題されます。有害業務に関する条文は第二種の出題範囲外です。たとえば残り10週の6月6日なら、関係法令10問演習で4問正解（40点）を第3週までに2回連続で取り、誤答論点を本記事の優先表と照合するのが定番です。本記事では10問向けの頻出論点を表に整理し、条文の骨格は関係法令基礎記事·数値は計算記事へ案内します。正本は労働安全衛生法·規則と安全衛生技術試験協会（https://www.exam.or.jp/）要項です。

この記事の要点

本記事を読むと、たとえば関係法令10問で「選任2問·健診3問·面接1問」と内訳を記録し、12テーマのうちどこから復習すべきかが優先順付きで分かります。

  • 関係法令10問
  • 12頻出テーマ
  • 条文·制度
  • 主体の義務
  • 有害業務は出題外

1関係法令10問と頻出論点の全体像

  • 関係法令は条文·制度·義務·届出が中心で
  • 10問すべてが独立した論点というより
  • 12前後のテーマから2問ずつ出るイメージで整理すると効率的

| 区分 | 内容 | 第二種 |
| --- | --- | --- |
| 出題数 | 10問·100点 | 30問中1科目 |
| 足切り | 4問正解（40点） | 各科目必須 |
| 論点 | 選任·健診·環境等 | 有害業務は出題外 |

たとえば関係法令10問演習で3問正解（30点）なら、本記事の優先テーマ表で★印2つ以上の論点から条文復習に戻ります。学習時間の配分は3か月計画記事で調整してください。

2優先度付き12テーマ一覧

関係法令ハブの12テーマを、演習頻度と足切りリスクで優先順を付けます。

| 優先 | テーマ | 条文·制度の焦点 |
| --- | --- | --- |
| ★★ | 衛生管理者選任 | 法第13条·50人の壁 |
| ★★ | 定期·特殊健康診断 | 頻度6ヶ月·1年 |
| ★★ | 長時間労働者面接指導 | 80時間·産業医 |
| ★ | 作業環境測定 | 測定·記録·評価 |
| ★ | 化学物質管理 | 規則の適用·表示 |
| ★ | 就業上の措置 | 事業者の義務 |
| | ストレスチェック | 面接指導と別制度 |
| | 安全衛生教育 | 実施者·記録 |
| | 衛生委員会 | 設置·議事録 |
| | 産業医 | 選任·契約 |
| | 届出·記録保存 | 3年·5年·30年 |
| | 監督·罰則 | 立入検査等 |

  • 例えば6月第1週は★★3テーマ
  • 第2週は★3テーマと
  • 週3論点×4週で12項目を一周し

各テーマの条文整理は関係法令基礎記事を参照してください。

3主体（事業者·衛生管理者·産業医）の横断表

頻出論点は「誰が何をするか」の主体確認とセットで出ます。

| 制度 | 事業者 | 衛生管理者 | 産業医 |
| --- | --- | --- | --- |
| 定期健康診断 | 実施義務 | 連携 | 結果の医学的管理 |
| 面接指導 | 就業措置 | 把握·連携 | 面接実施 |
| 作業環境測定 | 実施·記録 | 改善計画 | 助言 |
| 選任·届出 | 選任義務 | 職務 | 契約·助言 |

  • たとえば関係法令10問で面接指導2問中1問正解·「衛生管理者が面接する」肢を誤選したら
  • 上表の面接指導行だけを条文見出し付きで書き写し
  • 翌週の演習10問で同論点を解き直し

80時間基準の数値は関係法令計算記事で整理してください。

4テーマ別の典型肢パターン

頻出論点は次の肢パターンで繰り返されます。

| 論点 | 正答の方向 | 誤答の典型 |
| --- | --- | --- |
| 選任 | 人数基準どおり選任 | 50人未満で不要と断定 |
| 健診 | 事業者が実施 | 産業医が健診実施 |
| 保存 | 規定の年数 | 他制度の年数を混同 |
| 面接 | 産業医が実施 | 衛生管理者が面接 |
| 措置 | 事業者が就業措置 | 医師が配置転換を決定 |

  • 例えば関係法令10問で健診3問中2問正解·保存期間の肢で3年と5年を混同したら
  • 健診制度の「実施頻度」と「記録保存」を2列表に分け
  • 頻出数値リストの該当行と突合し

5演習で頻出論点を定着させる手順

頻出論点の定着は「テーマ分類→条文→演習」の3段階です。

1. 関係法令10問演習で誤答を12テーマ番号で分類
2. ★★テーマの条文を公式テキストで読み直す
3. 同テーマの一問一答5問で主体を確認
4. 40点以上が2回続いたら30問本番形式へ
5. 数値誤答は計算記事·数値リストで補強

たとえば第5週の関係法令10問で選任2問·健診3問·面接1問·作業環境2問·化学2問の内訳で4問正解（40点）なら、足切り合格圏ですが健診の保存期間を1問誤答したなら、No.2定期·特殊健康診断の条文復習を優先します。週次の時間配分は独学スケジュール記事を参照してください。

6よくある質問

12テーマは全部暗記が必要ですか？
全部を同じ深さで覚える必要はありません。★★の選任·健診·面接指導から始め、演習の誤答テーマを追加していく方式が現実的です。たとえば第1〜2週で★★3論点の条文を読み、第3週の関係法令10問で4問正解（40点）を目標にします。38点以下が2週続く場合は、数値暗記に週2時間を追加してください。
頻出論点と基礎記事の使い分けは？
基礎記事は条文·制度の骨格、本記事は演習前の優先順表です。例えば初学者は基礎記事で選任·健診の章を1周し、その後本記事の12テーマ表で演習の誤答位置を特定します。関係法令10問演習直前は本記事の★★行だけを10分で確認する使い方が効率的です。
ストレスチェックはどの程度出ますか？
面接指導ほど毎回ではありませんが、制度設計·実施主体·記録の違いとして出題されます。たとえば関係法令10問で面接指導2問正解·ストレス1問誤答なら、3列比較表（長時間·ストレス·深夜業）を作り、数値·主体·保存期間を過去問演習の誤答ノートに書き込みます。面接指導の詳細は面接指導80時間記事を参照してください。
